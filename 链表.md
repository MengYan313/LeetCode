## [链表的中间结点](https://leetcode.cn/problems/middle-of-the-linked-list/)(876)

问题：给你单链表的头结点 `head` ，请你找出并返回链表的中间结点。如果有两个中间结点，则返回第二个中间结点。

分析：双指针，`slow` 走一步 `fast` 走两步

* 偶数返回**第一个中间节点**：`fast.next != null && fast.next.next != null`
* 偶数返回**第二个中间节点**：`fast != null && fast.next != null`

复杂度：`O(n)` `O(1)`

```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    // fast.next != null && fast.next.next != null 偶数返回第一个中间节点
    while (fast != null && fast.next != null) { // 偶数返回第二个中间节点
        slow = slow.next; // 慢指针走一步
        fast = fast.next.next; // 快指针走两步
    }
    return slow; // 包括头节点为空或仅一个的情况，直接返回head
}
```

## [链表中倒数第K个节点](https://leetcode.cn/problems/lian-biao-zhong-dao-shu-di-kge-jie-dian-lcof/)(LCR140)

问题：给定一个头节点为 `head` 的链表用于记录一系列核心肌群训练项目编号，请查找并返回倒数第 `cnt` 个训练项目编号。

分析：双指针，`fast` 先走 `cnt - 1` 步，接着 `fast` 与 `slow` 同步前进

复杂度：`O(n)` `O(1)`

```java
public ListNode trainingPlan(ListNode head, int cnt) {
    ListNode slow = head, fast = head;
    for(int i = 0; i < cnt - 1; i++){ // 快指针先走cnt-1步
        fast = fast.next;
    }
    while(fast.next != null){ // fast指向最后一个节点，slow指向倒数第cnt个
        slow = slow.next;
        fast = fast.next;
    }
    return slow;
}
```

public int[] reverseBookList(ListNode head) {
    int len = 0;
    for(ListNode tmp = head; tmp != null; tmp = tmp.next){
        len++;

    int res[] = new int[len];
    for(ListNode tmp = head; tmp != null; tmp = tmp.next){
        res[--len] = tmp.val; // 遍历时倒序放入数组
    }
    return res;
}

## [相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/)(160)

问题：给你两个单链表的头节点 `headA` 和 `headB` ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null` 。

解法 1：双指针，先遍历各自链表再遍历另一条，第二次遍历时一定在相交处相遇

复杂度：`O(n + m)` `O(1)`

```java
public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
    ListNode a = headA, b = headB;
    while(a != b){ // 第二次遍历时一定在相交处相遇
        a = a != null ? a.next : headB; // 遍历完a接着遍历b
        b = b != null ? b.next : headA; // 遍历完b接着遍历a
    }
    return a; // a==b
}
```

解法 2：`set` 记录已遍历节点，第一个重复节点即为相交处

复杂度：`O(n + m)` `O(n)`

```java
public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
    Set<ListNode> visited = new HashSet<ListNode>();
    ListNode temp = headA;
    while(temp != null){
        visited.add(temp);
        temp = temp.next;
    }
    temp = headB;
    while(temp != null){
        if(visited.contains(temp)){
            return temp;
        }
        temp = temp.next;
    }
    return null;
}
```

## [环形链表](https://leetcode.cn/problems/linked-list-cycle/)(141)

问题：给你一个链表的头节点 `head` ，判断链表中是否有环。

分析：双指针，`slow` 走一步 `fast` 走两步，相遇则有环

复杂度：`O(n)` `O(1)`

```java
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) { // 需要保证fast的走两步不出错
        slow = slow.next;
        fast = fast.next.next; // fast.next != null
        if(fast == slow){
            return true;
        }
    }
    return false;
}
```

## [环形链表 II](https://leetcode.cn/problems/linked-list-cycle-ii/)(142)

问题：给定一个链表的头节点  `head` ，返回链表开始入环的第一个节点。 *如果链表无环，则返回 `null`。*

分析：双指针，`slow` 走一步 `fast` 走两步，相遇则有环。`slow` 和 `head` 同步前进，必定在入环处相遇。

复杂度：`O(n)` `O(1)`

```java
public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (fast == slow) {  // 若有环则快慢指针必然相遇
            while (slow != head) {  // a = c + (k - 1)(b + c) 慢指针与头指针必在入环处相遇
                slow = slow.next;
                head = head.next;
            }
            return slow;
        }
    }
    return null;
}
```

解法2：双指针，`slow` 走一步 `fast` 走两步，相遇则有环。统计环的节点个数为 `cnt`，重置双指针后 `fast` 先走 `cnt` 步，接着与 `slow` 同步前进，必定在入环处相遇。

复杂度：`O(n)` `O(1)`

```java
public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while(fast != null && fast.next != null){
        slow = slow.next;
        fast = fast.next.next;
        if(slow == fast){ // 相遇而退出，说明有环
            break;
        }
    }
    if(fast == null || fast.next == null){ // 不满足循环条件而退出，说明无环
        return null;
    }
    int cnt = 1; // 环中至少有一个节点
    for(fast = fast.next; slow != fast; fast = fast.next){ // 统计环的节点个数
        cnt++;
    }
    slow = head; fast = head; // 重置快慢指针
    for(int i = 0; i < cnt; i++){ // fast先走cnt步
        fast = fast.next;
    }
    while(true){ // fast多走一圈后与slow同步走，必定在入环处相遇
        if(slow == fast){
            return slow;
        }
        slow = slow.next;
        fast = fast.next;
    }
}
```

解法3：`set` 记录已遍历节点，第一个重复节点即为入环点

复杂度：`O(n)` `O(n)`

```java
public ListNode detectCycle(ListNode head) {
    Set<ListNode> visited = new HashSet<ListNode>();
    ListNode pos = head;
    while(pos != null){
        if(visited.contains(pos)){
            return pos;
        }
        else{
            visited.add(pos);
        }
        pos = pos.next;
    }
    return null;
}
```

## [链表随机节点](https://leetcode.cn/problems/linked-list-random-node/)(382)

问题：给你一个单链表，随机选择链表的一个节点，并返回相应的节点值。每个节点 **被选中的概率一样** 。

分析：统计长度 `n`，直接以 `1/n` 概率输出每个位置（需要遍历两次）

**蓄水抽样法**：在遍历时就决定是否输出当前的数，以 `1/i` 的概率选当前遍历到的数（仅需遍历一次）

复杂度：`O(n)` `O(1)`

```java
class Solution {
    ListNode head = null;
    Random random = null;

    public Solution(ListNode head) {
        this.head = head;
        this.random = new Random();
    }

    public int getRandom1() { //方法1 统计长度 O(n)=2n O(1)
        int n = 1;
        ListNode cur = head;
        while(cur.next != null){ // 统计链表长度
            n++;
            cur = cur.next;
        }
        int k = random.nextInt(n) + 1; // 直接以 1/n 概率输出每个位置
        cur = head;
        for(int i = 0; i < k - 1; i++){ // 找到随机到的位置在链表中的值
            cur = cur.next;
        }
        return cur.val;
    }

    public int getRandom() { //方法2 蓄水抽样法 O(n)=n O(1)
        int i = 1;
        int res = head.val;
        for(ListNode cur = head; cur != null; cur = cur.next){ // 在遍历时就决定是否输出当前的数
            if(random.nextInt(i) == 0){ // 1/i 的概率选当前遍历到的数
                res = cur.val; // 选到了就更新
            }
            i++;
        }
        return res; // 最终返回的每个值概率相等 1/2 * 2/3 * 3/4 = 1/4
    }
}
```

## [移除链表元素](https://leetcode.cn/problems/remove-linked-list-elements/)(203)

问题：给你一个链表的头节点 `head` 和一个整数 `val` ，请你删除链表中所有满足 `Node.val == val` 的节点，并返回 **新的头节点** 。

分析：`pre.next = cur.next` 完成删除操作，注意 `head == null` 先讨论

复杂度：`O(n)` `O(1)`

```java
public ListNode removeElements(ListNode head, int val) {
    while(head != null && head.val == val){ // 头节点没有前驱，特殊处理
        head = head.next;
    }
    if(head == null){ // 空链表或删完头节点后为空
        return head;
    }
    ListNode pre = head, cur = head.next;
    while(cur != null){
        if(cur.val == val){
            pre.next = cur.next; // 删除操作
        }else{
            pre = cur; // pre = pre.next;
        }
        cur = cur.next;
    }
    return head;
}
```

## [删除链表的倒数第 N 个结点](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/)(19)

问题：给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

分析：先统计链表长度，定位倒数第 `n + 1` 个节点进行删除，注意 `head == null` 先讨论

复杂度：`O(n)` `O(1)`

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    int len = 0;
    for(ListNode tmp = head; tmp != null; tmp = tmp.next){ // 统计链表长度
        len++;
    }
    if(len - n + 1 == 1){ // 头节点无前驱节点
        return head.next;
    }
    ListNode pre = head;
    for(int i = 0; i < len - n - 1; i++){ // (len - n + 1) - 1 - 1 找到前驱节点
        pre = pre.next;
    }
    pre.next = pre.next.next; // 删除节点操作
    return head;
}
```

解法2：使用 `dummyHead` 处理 `head == null`，使用[双指针](#[链表中倒数第k个节点](https://leetcode.cn/problems/lian-biao-zhong-dao-shu-di-kge-jie-dian-lcof/)(LCR140))定位倒数第 `n + 1` 个节点进行删除

复杂度：`O(n)` `O(1)`

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0, head); // 删除头节点需要创建哨兵节点dummy
    ListNode left = dummy, right = dummy;
    while (n-- > 0) { // 前指针比后指针多走n步
        right = right.next;
    }
    while (right.next != null) { // 找到倒数第n个节点的前一个节点
        left = left.next;
        right = right.next;
    }
    left.next = left.next.next; // 删除倒数第n个节点
    return dummy.next;
}
```

## [删除排序链表中的重复元素](https://leetcode.cn/problems/remove-duplicates-from-sorted-list/)(83)

问题：给定一个已排序的链表的头 `head` ， *删除所有重复的元素，使每个元素只出现一次* 。返回 *已排序的链表* 。

分析：判断下一个节点非空且和当前值重复，`cur.next = cur.next.next` 删除下一个重复节点

复杂度：`O(n)` `O(1)`

```java
public ListNode deleteDuplicates(ListNode head) {
    if (head == null) { // 可以保留头节点，不需要dummy
        return null;
    }
    ListNode cur = head;
    while (cur.next != null)
        if (cur.next.val == cur.val) { // 若下一个结点值重复则删除下一个结点
            cur.next = cur.next.next;
        } else { // 不重复则指针向前移一步
            cur = cur.next;
        }
    return head;
}
```

## [删除排序链表中的重复元素 II](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/)(82)

问题：给定一个已排序的链表的头 `head` ， *删除原始链表中所有重复数字的节点，只留下不同的数字* 。返回 *已排序的链表* 。

分析：设置 `dummy` 处理删除头节点，每次遍历下两个节点，若 `val` 相同则不断删除值为 `val` 的结点

复杂度：`O(n)` `O(1)`

public ListNode deleteDuplicatesII(ListNode head) {
    ListNode dummy = new ListNode(0, head), cur = dummy; // 由于需要删除所有的重复节点，则有可能删除头节点，故需要dummy
    while (cur.next != null && cur.next.next != null) { // 一次遍历两个结点
        int val = cur.next.val;
        if (cur.next.next.val == val) { // 若下两个结点值相同
            while (cur.next != null && cur.next.val == val) { // 循环删除值为 val 的结点
                cur.next = cur.next.next;
            }
        } else { // 若下两个结点值不同则指针向前移一步
            cur = cur.next;
        }
    }
    return dummy.next;

## [两数相加](https://leetcode.cn/problems/add-two-numbers/)(2)

问题：给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。请你将两个数相加，并以相同形式返回一个表示和的链表。

分析：同时顺序遍历并计算（若长度不一致则高位补0），用 `carry` 保存进位值，将每次计算结果**尾插**入新链表

复杂度：`O(Max(M,N))` `O(Max(M,N))`

```java
public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
    ListNode head = null, tail = null; // 定义新链表
    int carry = 0; // 进位值
    while (l1 != null || l2 != null) {
        int n1 = l1 != null ? l1.val : 0; // 若长度不一致则高位补0
        int n2 = l2 != null ? l2.val : 0;
        int sum = n1 + n2 + carry;
        if (head == null) { // 创建新链表头节点
            head = tail = new ListNode(sum % 10);
        } else { // 尾部插入新节点
            tail.next = new ListNode(sum % 10);
            tail = tail.next;
        }
        carry = sum / 10; // 更新进位值
        if (l1 != null) { // 若还有则继续向后遍历
            l1 = l1.next;
        }
        if (l2 != null) {
            l2 = l2.next;
        }
    }
    if (carry > 0) { // 最高位有进位则会多一位
        tail.next = new ListNode(carry);
    }
    return head;
}
```

## [合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists/)(21)

public ListNode mergeTwoLists(ListNode list1, ListNode list2) {

## [随机链表的复制](https://leetcode.cn/problems/copy-list-with-random-pointer/)(138)

问题：给你一个长度为 `n` 的链表，每个节点包含一个额外增加的随机指针 `random` ，该指针可以指向链表中的任何节点或空节点。

构造这个链表的 **[深拷贝](https://baike.baidu.com/item/深拷贝/22785317?fr=aladdin)**。 深拷贝应该正好由 `n` 个 **全新** 节点组成，其中每个新节点的值都设为其对应的原节点的值。新节点的 `next` 指针和 `random` 指针也都应指向复制链表中的新节点，并使原链表和复制链表中的这些指针能够表示相同的链表状态。**复制链表中的指针都不应指向原链表中的节点** 。

分析：先将原链表复制为 `A->A'->B->B'->C->C'`，再拆为原链表 `A->B->C` 和新链表 `A'->B'->C'`

复杂度：`O(n)` `O(1)`

```java
public Node copyRandomList(Node head) {
    if(head == null){
        return head;
    }
    Node cur = head, nxt = head;
    while(cur != null){ // 按next指针顺序复制一遍节点
        // nxt = cur.next;
        nxt = nxt.next; // 必须放循环里第一步，否则会 nxt == null
        cur.next = new Node(cur.val); // cur -> newNode
        cur.next.next = nxt; // newNode -> cur
        cur = nxt;
    }
    Node newCur;
    for(cur = head; cur != null; cur = cur.next.next){ // 在新节点上复制random指针
        newCur = cur.next; // 指向复制的新节点
        newCur.random = cur.random == null ? null : cur.random.next; // 复制
    }
    Node newHead = head.next; // 复制的第一个节点，待返回
    for(cur = head, newCur = head.next; cur != null; ){ // 拆分出新节点
        cur.next = cur.next.next; // 拆出原节点
        cur = cur.next; // 原节点指针前进
        // newCur.next = cur == null ? null : cur.next;
        newCur.next = cur == null ? null : newCur.next.next; // 注意cur前进后可能为null
        newCur = newCur.next; // 新节点指针前进
    }
    return newHead;
}
```

## [两两交换链表中的节点](https://leetcode.cn/problems/swap-nodes-in-pairs/)(24)

问题：给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。（不修改节点内部的值，只能进行节点交换）

分析：创建哑结点 `dummyHead`，令 `temp` 表示当前到达的节点，初始时 `temp = dummyHead`。每次需要交换 `temp` 后面的两个节点。

复杂度：`O(n)` `O(1)`

```java
public ListNode swapPairs(ListNode head) {
    ListNode dummyHead = new ListNode(); // 方便处理第一个表头节点
    dummyHead.next = head;
    ListNode temp = dummyHead;
    while(temp.next != null && temp.next.next != null){ // 如果 temp 后有两个节点，则进行交换
        ListNode node1 = temp.next; // 标记 node1
        ListNode node2 = temp.next.next; // 标记 node2
        node1.next = node2.next; // 后连接
        node2.next = node1; // 反转连接
        temp.next = node2; // 前连接
        temp = node1; // 向前遍历
    }
    return dummyHead.next;
}
```

## [反转链表](https://leetcode.cn/problems/reverse-linked-list/)(206)

问题：给你单链表的头节点 `head` ，请你反转链表，并返回反转后的链表。

解法 1：原地反转，`cur.next = pre`

复杂度：`O(n)` `O(1)`

```java
public ListNode reverseList(ListNode head) {
    ListNode pre = null, cur = head;
    while (cur != null) {
        ListNode nxt = cur.next;  // 记录当前结点的下一个，防止丢失
        cur.next = pre;  // 反转当前结点
        pre = cur;  // 指针后移
        cur = nxt;  // 指针后移
    }
    return pre;  // cur为空后，pre指向原链表的尾部
}
```

解法2：递归

* 函数定义：反转链表并返回头节点
* 边界条件：单个节点直接返回本身
* 转移关系：递归到下一个节点，再拼接

复杂度：`O(n)` `O(1)`

```java
public ListNode reverseList(ListNode head) { // 定义为反转链表并返回头节点
    if(head == null || head.next == null){ // 非空链表只会递归到 head.next == null
        return head;
    }
    ListNode newHead = reverseList(head.next); // 减少节点以缩小规模
    head.next.next = head; // 2->1
    head.next = null; // 1->null
    return newHead;
}
```

## [反转链表 II](https://leetcode.cn/problems/reverse-linked-list-ii/)(92)

问题：给你单链表的头指针 `head` 和两个整数 `left` 和 `right` ，其中 `left <= right` 。请你反转从位置 `left` 到位置 `right` 的链表节点，返回 **反转后的链表**

分析：原地反转，`cur.next = pre`

复杂度：`O(n)` `O(1)`

```java
public ListNode reverseBetween(ListNode head, int left, int right) {
    ListNode dummy = new ListNode(0, head), p0 = dummy; // 设置哨兵结点防止p0不存在
    for (int i = 0; i < left - 1; ++i) // p0指向反转前的最后一个节点
        p0 = p0.next;
    ListNode pre = null, cur = p0.next;
    for (int i = 0; i < right - left + 1; i++) { // 反转中间段
        ListNode nxt = cur.next; // 记录当前结点的下一个，防止丢失
        cur.next = pre; // 反转当前结点
        pre = cur; // 指针后移
        cur = nxt; // 指针后移
    }
    // 反转结束后，pre指向反转段的开头，cur指向反转段后续的下个节点
    p0.next.next = cur; // 连接反转段与尾段
    p0.next = pre; // 连接头段与反转段
    return dummy.next;
}
```

解法2：递归

* 函数定义：反转链表中第 `[1, n]` 个节点，并返回头节点
* 边界条件：单个节点直接返回本身，同时记录后续无需反转的链表部分
* 转移关系：递归到下一个节点，再拼接

复杂度：`O(n)` `O(1)`

```java
class Solution {
    ListNode last = null;

    // 反转以head为头节点的链表中，第[left,right]个节点
    public ListNode reverseBetween(ListNode head, int left, int right) {
        if(left == 1){ // 不断递归直至left为左端点
            return reverseThN(head, right);
        }
        // 减少节点以缩小规模，在以head.next为头节点的新链表中，left和right的值都减1
        ListNode newHead = reverseBetween(head.next, left - 1, right - 1);
        // head必须指向反转后链表的newHead(4)，否则会丢失反转的节点（默认指向的left(2)反转后在最后）
        head.next = newHead; // left左边的节点无需反转，直接连接即可
        return head;
    }

    // 反转以head为头节点的链表中，第[1,n]个节点
    public ListNode reverseThN(ListNode head, int n){
        if(n == 1){ // 仅剩一个节点无需反转
            last = head.next; // 记录后续无需反转的链表部分
            return head;
        }
        ListNode newHead = reverseThN(head.next, n - 1); // 减少节点以缩小规模
        head.next.next = head; // 接上头节点前驱
        head.next = last; // 接上头节点后继
        return newHead;
    }
}
```

## [K 个一组翻转链表](https://leetcode.cn/problems/reverse-nodes-in-k-group/)(25)

问题：给你链表的头节点 `head` ，每 `k` 个节点一组进行翻转，请你返回修改后的链表。`k` 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 `k` 的整数倍，那么请将最后剩余的节点保持原有顺序。你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

分析：原地反转，`cur.next = pre`，使用 `dummyHead` 处理 `head == null`

复杂度：`O(n)` `O(1)`

```java
public ListNode reverseKGroup(ListNode head, int k) {
    int n = 0;
    for (ListNode cur = head; cur != null; cur = cur.next) // 统计节点个数，判断结束条件
        n++;
    ListNode dummy = new ListNode(0, head), p0 = dummy;  // 设置哨兵结点防止p0不存在
    ListNode pre = null, cur = head;
    for (; n >= k; n -= k) {
        for (int i = 0; i < k; i++) { // K 个一组翻转链表
            ListNode nxt = cur.next;  // 记录当前结点的下一个，防止丢失
            cur.next = pre;  // 反转当前结点
            pre = cur;  // 指针后移
            cur = nxt;  // 指针后移
        }
        ListNode nxt0 = p0.next;  // 记录p0的下一个，防止丢失
        p0.next.next = cur;  // 连接反转段与尾段
        p0.next = pre;  // 连接头段与反转段
        p0 = nxt0;  // 更新p0，为下一组反转做准备
    }
    return dummy.next;
}
```

解法2：递归：按组反转不断缩小问题规模，为此需解决反转一个子链表的问题 [`reverseList`](#[反转链表](https://leetcode.cn/problems/reverse-linked-list/)(206))

* 函数定义：`reverseKGroup:` `k` 个一组反转链表，并返回头节点
* 边界条件：当本组节点数量不足 `k` 个时，无需反转，直接返回头节点
* 转移关系：递归到反转下一组节点，再拼接

复杂度：`O(n)` `O(n/k + k)`

```java
public ListNode reverseKGroup(ListNode head, int k) { // 定义为以k个一组反转链表
    ListNode tmp = head; // tmp初始化指向每组的头节点
    for(int i = 0; i < k - 1 && tmp != null; i++){ // 注意保证tmp不能为空
        tmp = tmp.next; // 循环k-1次，使得tmp指向每组最后一个节点
    }
    if(tmp == null){ // tmp为空说明这一组节点不足k个，不反转直接返回head
        return head;
    }
    ListNode nxtHead = tmp.next; // nxtHead指向下一组节点的第一个
    tmp.next = null; // 切割，指向null以表明一个子链表的结束（否则反转链表递归函数不知道何时停止）
    ListNode newHead = reverseList(head); // 反转本组节点
    ListNode newTmp = reverseKGroup(nxtHead, k); // 将反转下一组节点作为下一个子问题
    head.next = newTmp; // 拼接，本组的原头节点接上下一组的第一个
    return newHead;
}

public ListNode reverseList(ListNode head) { // 定义为反转链表并返回头节点
    if(head == null || head.next == null){ // 非空链表只会递归到 head.next == null
        return head;
    }
    ListNode newHead = reverseList(head.next); // 减少节点以缩小规模
    head.next.next = head; // 2->1
    head.next = null; // 1->null
    return newHead;
}
```

## [旋转链表](https://leetcode.cn/problems/rotate-list/)(61)

问题：给你一个链表的头节点 `head` ，旋转链表，将链表每个节点向右移动 `k` 个位置。

分析：先成环（顺便获取链表长度 `n`），再拆开（第 `n - k % n` 个节点处拆开）

复杂度：`O(n)` `O(1)`

```java
public ListNode rotateRight(ListNode head, int k) { // 先成环，再拆开
    if(head == null){
        return head;
    }
    int n = 1; // 去除空链表的情况，至少有一个节点
    ListNode tmp = head;
    while(tmp.next != null){ // 统计节点数量，并使得tmp指向最后一个节点
        n++;
        tmp = tmp.next;
    }
    tmp.next = head; // 成环
    for(int i = 0; i < n - k % n - 1; i++){ // 在第 n - k % n 个节点处拆开
        head = head.next; // 头节点向前走 n - k % n - 1 步
    }
    ListNode newHead = head.next; // 记录旋转后的新头节点
    head.next = null; // 在指定位置拆开
    return newHead;
}
```

## [排序链表](https://leetcode.cn/problems/sort-list/)(148)

问题：给你链表的头结点 `head` ，请将其按 **升序** 排列并返回 **排序后的链表** 。

分析：递归实现归并排序，并且利用辅助函数 [`middleNode`](#[链表的中间结点](https://leetcode.cn/problems/middle-of-the-linked-list/)(876)) 和 [`mergeTwoLists`](#[合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists/)(21))

* 函数定义：归并排序链表，并返回头节点
* 边界条件：单个节点直接返回本身（`middleNode`）
* 转移关系：根据中间节点分为两个子列表，分别递归进行归并排序，再合并两个有序列表（`mergeTwoLists`）

复杂度：`O(nlog(n))` `O(log(n))`

```java
public ListNode sortList(ListNode head) { // 归并排序
    if(head == null || head.next == null){ // 递归边界条件
        return head;
    }
    ListNode mid = middleNode(head); // 获取第一个中间节点（第二个不好拆分）
    ListNode left = head, right = mid.next; // 将链表分为左右两段
    mid.next = null; // 左链表断开
    ListNode leftHead = sortList(left); // 递归左右链表进行归并排序
    ListNode rightHead = sortList(right);
    head = mergeTwoLists(leftHead, rightHead); // 合并两个有序链表
    return head; // 也可以在上一步用新节点mergeHead接受，逻辑更清晰
}

public ListNode middleNode(ListNode head) { // 返回链表的中间节点
    ListNode slow = head, fast = head;
    while(fast.next != null && fast.next.next != null){ // 偶数返回第一个节点
        // while (fast != null && fast.next != null) { // 偶数返回第二个节点
        slow = slow.next; // 慢指针走一步
        fast = fast.next.next; // 快指针走两步
    }
    return slow; // 包括头节点为空或仅一个的情况，直接返回head
}

public ListNode mergeTwoLists(ListNode list1, ListNode list2) { // 合并两个有序链表
    ListNode dummyHead = new ListNode(); // 虚拟头节点简化判断，并定义指向空节点的非空指针
    ListNode merge = dummyHead; // 若merge=null空指针，则需merge=list1的赋值方式
    while (list1 != null && list2 != null) { // 双指针遍历
        if (list1.val < list2.val) {
            merge.next = list1;
            list1 = list1.next;
        } else {
            merge.next = list2;
            list2 = list2.next;
        }
        merge = merge.next;
    }
    merge.next = list1 != null ? list1 : list2; // 将剩余的某段链表直接接上
    return dummyHead.next; // 返回虚拟头节点的下一个节点
}
```

## [重排链表](https://leetcode.cn/problems/reorder-list/)(143)

问题：给定一个单链表 `L` 的头节点 `head` ，单链表 `L` 表示为：

```
L0 → L1 → … → Ln-1 → Ln
```

请将其重新排列后变为：（不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换）

```
L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → …
```

分析：先**反转后半段链表**，再和前半段链表依次插入即可

* 为了反转后半段链表，首先要获取链表的中间结点 `mid`，利用辅助函数 [`middleNode`](#[链表的中间结点](https://leetcode.cn/problems/middle-of-the-linked-list/)(876))，并将 `mid` 作为后半段链表的头节点
* 其次利用辅助函数 [`reverseList`](#[反转链表](https://leetcode.cn/problems/reverse-linked-list/)(206)) 反转后半段链表，并用 `head2` 接收新表头（原表尾）

复杂度：`O(n)` `O(1)`

```java
public void reorderList(ListNode head) {
    ListNode mid = middleNode(head); // 为了反转后半段链表，首先要获取链表的中间结点
    ListNode head2 = reverseList(mid); // 相向双指针，为了让head2移动首先要反转后半段链表，反转后 mid.next==null
    while (head2.next != null) { // 偶数时一样长，奇数时head2多一个节点
        ListNode nxt = head.next; // 记录head
        ListNode nxt2 = head2.next; // 记录head2
        head.next = head2; // 由小到大连接
        head2.next = nxt; // 由大到小连接
        head = nxt; // 更新head
        head2 = nxt2; // 更新head2
    }
}

public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    // fast.next != null && fast.next.next != null 偶数返回第一个中间节点
    while (fast != null && fast.next != null) { // 偶数返回第二个中间节点
        slow = slow.next; // 慢指针走一步
        fast = fast.next.next; // 快指针走两步
    }
    return slow; // 包括头节点为空或仅一个的情况，直接返回head
}

public ListNode reverseList(ListNode head) {
    ListNode pre = null, cur = head;
    while (cur != null) {
        ListNode nxt = cur.next;  // 记录当前结点的下一个，防止丢失
        cur.next = pre;  // 反转当前结点
        pre = cur;  // 指针后移
        cur = nxt;  // 指针后移
    }
    return pre;  // cur为空后，pre指向原链表的尾部
}
```
