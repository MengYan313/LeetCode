# 位运算

## [只出现一次的数字](https://leetcode.cn/problems/single-number/)(136)

问题：给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

分析：异或运算：`a ^ b ^ b == a` `a ^ 0 == a`

复杂度：`O(n)` `O(1)`

```java
public int singleNumber(int[] nums) {
    int ans = 0; // a ^ 0 == a
    for(int i = 0; i < nums.length; i++){
        ans = ans ^ nums[i]; // a ^ b ^ b == a
    }
    return ans;
}
```

## [只出现一次的数字 II](https://leetcode.cn/problems/single-number-ii/)(137)

问题：给你一个整数数组 `nums` ，除某个元素仅出现 **一次** 外，其余每个元素都恰出现 **三次 。**请你找出并返回那个只出现了一次的元素。

分析：记录 `nums` 里所有数的二进制在每一位上 `1` 的个数，再对 `3` 取模即可

复杂度：`O(n)` `O(1)`

```java
public int singleNumber(int[] nums) {
    int res[] = new int [32]; // 记录nums里所有数的二进制在每一位上1的个数，再对3取模即可
    int m = 1, sum = 0;
    for(int i = 0; i < 32; i++){ // 按位计算1的个数
        for(int j = 0; j < nums.length; j++){
            if((nums[j] & m) != 0){ // 当前第i位是1，注意不能写成 (nums[j] & m) == 1
                res[i]++;
            }
        }
        res[i] %= 3; // 1/0，表示唯一出现的元素在第i位上的取值
        sum += res[i] * m; // 也可得到res数组后在计算
        m = m << 1; // 左移一位，准备计算下一位
    }
    return sum;
}
```

## [只出现一次的数字 III](https://leetcode.cn/problems/single-number-iii/)(260)

问题：给你一个整数数组 `nums`，其中恰好有两个元素只出现一次，其余所有元素均出现两次。 找出只出现一次的那两个元素。你可以按 **任意顺序** 返回答案。

分析：异或运算：`a ^ b ^ b == a` `a ^ 0 == a`

* 先对 `nums` 所有元素异或使得 `z == x ^ y`
* 再用 `m` 表示第几个二进制位不同，即 `z` 的第 `m` 位是 `1`（找到任意一位 `x` 与 `y` 不同即可）
* 从而将 `nums` 分为两个数组， `x` 与 `y` 分别分布在两个数组里，转化为[只出现一次的数字(136)](#[只出现一次的数字](https://leetcode.cn/problems/single-number/)(136))，分别进行异或运算即可

复杂度：`O(n)` `O(1)`

```java
public int[] singleNumber(int[] nums) {
    int x = 0, y = 0, z = 0;  // 对数组所有元素异或使得 z == x ^ y
    for(int i = 0; i < nums.length; i++){
        z = z ^ nums[i];
    }
    int m = 1; // m表示第几个二进制位不同，即z的第m位是1（找到任意一位x与y不同即可，将其分为两个数组）
    while((m & z) == 0){ // z的第m位是1则退出循环，记下m
        m = m << 1; // 向左移动一位 1000...
    }
    for(int i = 0; i < nums.length; i++){
        if((nums[i] & m) == 0){ // 不能写成 (nums[i] & m) == 1，因为 m == 0...010...0, false
            x = x ^ nums[i]; // 不需要额外new一个数组，直接计算就行
        }else{
            y = y ^ nums[i];
        }
    }
    return new int[]{x, y};
}
```

## [位 1 的个数](https://leetcode.cn/problems/er-jin-zhi-zhong-1de-ge-shu-lcof/)(LCR133)

问题：编写一个函数，输入是一个无符号整数（以二进制串的形式），返回其二进制表达式中数字位数为 '1' 的个数（ [汉明重量](http://en.wikipedia.org/wiki/Hamming_weight)）。

分析：`n & (n - 1)` 将 `n` 二进制最右端的 `1` 转为 `0`，同时统计数量即可

复杂度：`O(logn)` `O(1)`

```java
public int hammingWeight(int n) {
    int sum = 0;
    while(n != 0){
        n = n & (n - 1);
        sum++;
    }
    return sum;
}
```

## 快速幂(x的n次方)

问题：计算 `x` 的 `n` 次方，均为正整数

分析：利用指数 `n` 的二进制表示，每次先 `n & 1 == 1` 取最低位的值，再 `n = n >> 1` 右移 `1` 位直至为 `0`

复杂度：`O(logn)` `O(1)`

```java
public long pow(long x, long n){
    long res = 1;
    while(n > 0){
        if((n & 1) == 1){ // 取最低位的值，并判断是否为1
            res *= x;
        }
        x *= x;
        n = n >> 1; // 右移一位 n %= 2
    }
    return res;
}

public long pow(long x, long n, long mod){
    long res = 1;
    while(n > 0){
        if((n & 1) == 1){ // 取最低位的值，并判断是否为1
            res = res * x % mod;
        }
        x = x * x % mod;
        n = n >> 1; // 右移一位 n %= 2
    }
    return res;
}
```

## 最大异或值

问题：在数组 `nums` 中选择两个数，使得异或值最大

分析：**构建字典树：每个数从二进制的最高位到最低位依次插入，左右节点分别表示0和1**

先将第一个数插入字典树，从第二个数开始与当前字典树计算最大异或值：从当前数最高位到最低位在字典树中查找**相反位**（尽量选择异 或值为 `1` 的路径），得到当前数和前面所有数能计算出的最大异或值。最后再将当前数插入字典树，迭代查找下一个数。

```java
static class TrieNode { // 定义字典树的节点
    TrieNode[] children;

    TrieNode() {
        children = new TrieNode[2]; // 0和1两种可能
    }
}

public static void insert(TrieNode root, int num) { // 插入一个数到字典树中
    TrieNode node = root;
    for (int i = 31; i >= 0; i--) { // 从最高位（31位）到最低位（0位）
        int bit = (num >>> i) & 1; // 无符号右移左侧始终填充 0，更明确地表示我们是在处理二进制位，而不关心数值的符号
        if (node.children[bit] == null) { // 如果当前位的子节点不存在，则创建一个新的节点
            node.children[bit] = new TrieNode();
        }
        node = node.children[bit]; // 移动到当前位对应的子节点
    }
}

public static int findMaxXOR(TrieNode root, int num) { // 在字典树中查找与num异或最大的值
    int maxXOR = 0;
    for (int i = 31; i >= 0; i--) {
        int bit = (num >>> i) & 1;
        int toggledBit = bit ^ 1; // 尽量选择相反的位，以最大化异或值
        if (root.children[toggledBit] != null) { // 如果相反的位存在,设置对应位的异或结果为1,并移动到相反位的子节点
            maxXOR |= (1 << i);
            root = root.children[toggledBit];
        } else { // 否则，选择相同的位，移动到对应的子节点
            root = root.children[bit];
        }
    }
    return maxXOR;
}

public static int findMaximumXOR(int[] nums) { // 主函数，计算数组中两个数的最大异或值
    if (nums == null || nums.length < 2) {
        return -1; // 数组中至少需要两个数
    }
    TrieNode root = new TrieNode();
    insert(root, nums[0]); // 将第一个数插入字典树
    int maxResult = 0; // 从第二个数开始，依次计算当前数与字典树中数的最大异或值，并将当前数插入字典树
    for (int i = 1; i < nums.length; i++) {
        int currentMax = findMaxXOR(root, nums[i]);
        maxResult = Math.max(maxResult, currentMax);
        insert(root, nums[i]);
    }
    return maxResult;
}
```

## [多数元素](https://leetcode.cn/problems/majority-element/)(169)

问题：给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。

分析：摩尔投票法（`Moore's Voting Algorithm`），先初始化一个候选元素 `candidate` 和计数器 `count`

* 如果 `x` 等于 `candidate`，则 `count` 增加 `1`；否则 `count` 减少 `1`
* 如果 `count == 0`，则将当前元素设为候选元素 `candidate = nums[i]`，并设置 `count = 1`

复杂度：`O(n)` `O(1)`

```java
public int majorityElement(int[] nums) { // 摩尔投票法（Moore's Voting Algorithm）
    int candidate = nums[0]; // 初始化一个候选元素 candidate 和计数器 count
    int count = 1;
    for (int i = 1; i < nums.length; i++) {
        if (nums[i] == candidate) { // 如果 x 等于 candidate，则 count 增加 1；否则 count 减少 1。
            count++;
        } else {
            count--;
        }
        if (count == 0) { // 如果 count 为 0，则将当前元素 x 设为候选元素 candidate，并将 count 设为 1
            candidate = nums[i];
            count = 1;
        }
    }
    return candidate;
}
```

## [整数拆分](https://leetcode.cn/problems/integer-break/)(343)

问题：给定一个正整数 `n` ，将其拆分为 `k` 个 **正整数** 的和（ `k >= 2` ），并使这些整数的乘积最大化。

分析：对于大于 `4` 的正整数，总是存在一种拆分的方案，使得拆分成的两个正整数的乘积大于拆分前的正整数

* 若 `f >= 4`，`2 * (f - 2) = 2 * f - 4 >= f`，所以 `>= 4` 的数字都可拆，又因为拆 `1` 没意义，所以拆为 `2` 或 `3`
* `3 * 3 > 2 * 2 * 2` 同样是 `6` 拆 `3 + 3` 值更大，所以优先拆 `3`，余数凑 `2`，拆除的 `2` 的数量需要 `< 3`

复杂度：`O(1)` `O(1)`

```java
public int integerBreak(int n) {
    if (n <= 3) {
        return n - 1;
    }
    int quotient = n / 3;
    int remainder = n % 3;
    if (remainder == 0) {
        return (int) Math.pow(3, quotient);
    } else if (remainder == 1) {
        return (int) Math.pow(3, quotient - 1) * 2 * 2;
    } else {
        return (int) Math.pow(3, quotient) * 2;
    }
}
```

## [整数拆分取模](https://leetcode.cn/problems/jian-sheng-zi-ii-lcof/)(LCR132)

问题：给定一个正整数 `n` ，将其拆分为 `k` 个 **正整数** 的和（ `k >= 2` ），并使这些整数的乘积最大化。

分析：对于大于 `4` 的正整数，优先拆 `3`，余数凑 `2`，同[整数拆分](#[整数拆分](https://leetcode.cn/problems/integer-break/)(343))，同时处理取模：

* `(xy)⊙p=[(x⊙p)(y⊙p)]⊙p` `->` 循环求余：`x`^n^`⊙p=[(x`^(n−1)^`⊙p)(x⊙p)]⊙p=[(x`^(n-1)^`⊙p)*x]⊙p`
* 定义 `res`~n~` = x`^n^`⊙p`，则计算 `pow(int x, int n)` 时有 `res`~n~` = res`~n-1~` * x % p`
  * 由于 `res * x` 可能溢出，故定义 `res` 的数据类型为 `long`
  * 由于 `pow(3, quotient - 1) * 2 * 2` 和 `pow(3, quotient) * 2` 可能溢出，故需取模 `% p`，此外需将 `pow()` 的返回类型定义为 `long`，最后还需进行类型转换 `(int)(pow(3, quotient) * 2 % p)`

复杂度：`O(logn)` `O(1)`

```java
public int cuttingBamboo(int bamboo_len) {
    int n = bamboo_len;
    int p = (int)(1e9 + 7); // int p = 1000000007;
    if (n <= 3) {
        return n - 1;
    }
    int quotient = n / 3;
    int remainder = n % 3;
    if (remainder == 0) {
        return (int)pow(3, quotient);
    } else if (remainder == 1) {
        return (int)(pow(3, quotient - 1) * 2 * 2 % p); // *4后可能溢出，需要取模
    } else {
        return (int)(pow(3, quotient) * 2 % p); // *2后可能溢出，需要取模
    }
}

// (xy)⊙p=[(x⊙p)(y⊙p)]⊙p -> 循环求余：x^n⊙p=[(x^(n−1)⊙p)(x⊙p)]⊙p=[(x^(n-1)⊙p)*x]⊙p
public long pow(int x, int n){ // *4/2后可能溢出，需要设为long
    int p = (int)(1e9 + 7); // int p = 1000000007;
    long res = 1; // res * x 可能溢出，设为long
    for(int i = 0; i < n; i++){ // 循环求余：res(n) = res(n-1) * x % p
        res = res * x % p;
    }
    return res;
}
```

## [k 镜像数字的和](https://leetcode.cn/problems/sum-of-k-mirror-numbers/)(2081)

问题：一个 **k 镜像数字** 指的是一个在十进制和 k 进制下从前往后读和从后往前读都一样的 **没有前导 0** 的 **正** 整数。比方说，`9` 是一个 2 镜像数字。`9` 在十进制下为 `9` ，二进制下为 `1001` ，两者从前往后读和从后往前读都一样。相反地，`4` 不是一个 2 镜像数字。`4` 在二进制下为 `100` ，从前往后和从后往前读不相同。给你进制 `k` 和一个数字 `n` ，请你返回 k 镜像数字中 **最小** 的 `n` 个数 **之和** 。

分析：折半搜索：区域生成枚举十进制镜像数：`1~9 -> 10~99`    `10~99 -> 101~9999`

* 奇数回文（生成镜像时不保留末位）：`10 -> 101`   `99 -> 999`
* 偶数回文（生成镜像时保留末位）：`10 -> 1001`   `99 -> 9999`

```java
private boolean isPalindrome(long x, int k) { // 判断数字是否是k进制下的回文数
    StringBuilder digit = new StringBuilder(); // 将x转换为k进制并判断回文
    while (x > 0) {
        digit.append(x % k);
        x /= k;
    }
    String str = digit.toString();
    return str.equals(digit.reverse().toString()); // 判断字符串是否回文
}

public long kMirror(int k, int n) {
    long ans = 0;
    int cnt = 0;
    int left = 1; // 设置初始区域起点是1
    while (cnt < n) {
        int right = left * 10; // 设置区域终点是起点10倍，即枚举 [1, 10) = [1, 9]
        for (int op = 0; op <= 1; op++) { // op 为 0 表示生成奇数回文，1 表示生成偶数回文
            for (int i = left; i < right; i++) { // 区域枚举 [left, right)
                if (cnt == n) {
                    break;
                }
                long combined = i; // 初始化回文串前半段
                long x = (op == 0) ? (i / 10) : i; // 若是奇数回文，生成镜像时不保留末位
                while (x > 0) { // 拼接 combined 的后半段，使其成为回文串
                    combined = combined * 10 + (x % 10);
                    x /= 10;
                }
                if (isPalindrome(combined, k)) { // 如果在k进制下也是回文数，找到一个答案
                    cnt++;
                    ans += combined;
                }
            }
        }
        left = right; // 当前区域枚举完毕后，下一个区域起点是当前区域终点+1（注意左开右闭）
    }
    return ans;
}
```

## [Z 字形变换](https://leetcode.cn/problems/zigzag-conversion/)(6)

问题：将一个给定字符串 `s` 根据给定的行数 `numRows` ，以从上往下、从左到右进行 Z 字形排列。比如输入字符串为 `"PAYPALISHIRING"` 行数为 `3` 时，排列如下：

```
P   A   H   N
A P L S I I G
Y   I   R
```

之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比如：`"PAHNAPLSIIGYIR"`。

分析：使用 `flag = -1 / 1` 表示向后/前走，走到第一行或最后一行则反向 `flag = - flag`

复杂度：`O(n)` `O(n)`

```java
public String convert(String s, int numRows) {
    if(numRows < 2){
        return s;
    }
    List<StringBuffer> rows = new ArrayList<>();  // 从左至右存储第i行的字符
    for(int i = 0; i < numRows; i++){  // 初始化 ArrayList，每个元素都是一个空 StringBuffer 对象
        rows.add(new StringBuffer());
    }
    int i = 0, flag = -1;  // flag标记用来反向
    for(char c : s.toCharArray()) {
        rows.get(i).append(c); // 第 i 行加入字符 c
        if(i == 0 || i == numRows -1){  // 走到第一行或最后一行则反向
            flag = - flag;
        }
        i += flag;  // 再走一步
    }
    StringBuffer res = new StringBuffer();
    for(StringBuffer row : rows){  // 依次按行输出最终结果
        res.append(row);
    }
    return res.toString();
}
```

## [最长公共前缀](https://leetcode.cn/problems/longest-common-prefix/)(14)

问题：编写一个函数来查找字符串数组中的最长公共前缀。如果不存在公共前缀，返回空字符串 `""`。

分析：以第一个字符串为基准，依次比较其他字符串的第 `i` 位是否与第一个字符串相同

复杂度：`O(nm)` `O(1)`

```java
public String longestCommonPrefix(String[] strs) {
    if (strs == null || strs.length == 0) {  // 任选其一即可
        return "";
    }
    int length = strs[0].length();  // 以第一个字符串为基准比较
    int count = strs.length;
    for (int i = 0; i < length; i++) {
        char c = strs[0].charAt(i);  // 依次获取第一个字符串的字符
        for (int j = 1; j < count; j++) {  // 依次比较其他字符串的第 i 位是否与第一个字符串相同
            if (i == strs[j].length() || strs[j].charAt(i) != c) {  // 若某字符串溢出或字符不等，返回前一次的值
                return strs[0].substring(0, i);  // 其实是截取[0,i)
            }
        }
    }
    return strs[0];  // 第一个字符串全部比较完毕，则最长公共前缀即为第一个字符串
}
```

解法2：对 `strs` 进行排序，仅需比较第一个与最后一个字符串即可得出 `LCP`

复杂度：`O(nmlogn)` `O(m)`

```java
public String longestCommonPrefix(String[] strs) {
    if (strs == null || strs.length == 0) {
        return "";
    }
    Arrays.sort(strs);
    StringBuilder ans = new StringBuilder();
    String first = strs[0], last = strs[strs.length - 1]; // 获取排序后第一个和最后一个字符串
    for (int i = 0; i < first.length(); i++) { // 排序后 first.length() <= last.length()
        if (first.charAt(i) == last.charAt(i)) { // 依次比较第 i 位，相同则属于LCP
            ans.append(first.charAt(i));
        } else {
            break;
        }
    }
    return ans.toString();
}
```

## [下一个排列](https://leetcode.cn/problems/next-permutation/)(31)

问题：整数数组的一个 **排列** 就是将其所有成员以序列或线性顺序排列。整数数组的 **下一个排列** 是指其整数的下一个字典序更大的排列。更正式地，如果数组的所有排列根据其字典顺序从小到大排列在一个容器中，那么数组的 **下一个排列** 就是在这个有序容器中排在它后面的那个排列。如果不存在下一个更大的排列，那么这个数组必须重排为字典序最小的排列（即，其元素按升序排列）。给你一个整数数组 `nums` ，找出 `nums` 的下一个排列。必须**[ 原地 ](https://baike.baidu.com/item/原地算法)**修改，只允许使用额外常数空间。

分析：为了保持更新后最小，**应保持尽可能多的高数位不变，并且从最低的数位（个位）逐渐向高位（左）遍历，易发现降序情况下不可更新需要再往前找一位，故先需要找到第一个升序序列**。

算法：**从后向前找到第一个升序序列**作为更新的最高位 `i`，将其**换成低位中第一个`> nums[i]`**的数字，再将**所有低位升序排列**即可

复杂度：`O(n)` `O(1)`

```java
public void nextPermutation(int[] nums) {
    int i, n = nums.length;
    for (i = n - 2; i >= 0 && nums[i] >= nums[i + 1]; i--) { }  // 首先从后向前查找第一个递增顺序对 (i,i+1) 若无则 i 降至-1
    if (i >= 0) {  // 若存在下一个排列，需要先更新最高位
        int j;
        for (j = n - 1; nums[j] <= nums[i]; j--) {  // 在区间 [i+1...n-1] 中从后向前查找第一个大于现有的数字
        }
        int temp = nums[i];  // 交换 nums[i] 与 nums[j] ，实现最高位的更新
        nums[i] = nums[j];
        nums[j] = temp;
    }
    // 此时 nums[i+1...n-1] 必为降序，需要将其转为升序
    Arrays.sort(nums, i + 1, n);  // Arrays.sort()对数组排序，区间为 [i+1,n)
}
```

## [旋转图像](https://leetcode.cn/problems/rotate-image/)(48)

问题：给定一个 *n* × *n* 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。你必须在**[ 原地](https://baike.baidu.com/item/原地算法)** 旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要** 使用另一个矩阵来旋转图像。

解法1：使用辅助数组：顺时针旋转`90°`：对于矩阵第 `i` 行的第 `j` 个元素，旋转后在倒数第 `i` 列的第 `j` 个位置。即 `matrix[i][j] -> matrix[j][n - 1 - i]`

复杂度：`O(n`^2^`)` `O(n`^2^`)`

```java
public void rotate(int[][] matrix) {
    int n = matrix.length;
    int[][] matrix_new = new int[n][n];
    for (int i = 0; i < n; i++) {  // 构造旋转后的新矩阵
        for (int j = 0; j < n; j++) {
            matrix_new[j][n - 1 - i] = matrix[i][j];
        }
    }
    for (int i = 0; i < n; i++) {  // 将新矩阵复制到原矩阵
        for (int j = 0; j < n; j++) {
            matrix[i][j] = matrix_new[i][j];
        }
    }
}
```

解法2：原地旋转（**分块旋转**）：根据 `matrix[i][j] -> matrix[j][n - 1 - i]` 不断迭代，即令 `i' = j, j' = n - 1 - j` 迭代，`(i, j) -> (j, n - 1 - i) -> (n - 1 - i, n - 1 - j) -> (n - 1 - j, i) -> (i, j)`，**迭代四次后回到原点，因此只需将矩阵划分为四部分，对第一部分施行以上变换**即可

* `n` 为偶数时，`i` 需要枚举 `[0...n/2-1]`，`j` 需要枚举 `[0...n / 2 - 1] = [0...(n + 1) / 2 - 1]`
* `n` 为奇数时，`i` 需要枚举 `[0...n/2-1]`，`j` 需要枚举 `[0...n / 2]     = [0...(n + 1) / 2 - 1]`

复杂度：`O(n`^2^`)` `O(1)`

```java
public void rotate2(int[][] matrix) {
    int n = matrix.length;
    for (int i = 0; i <= n / 2 - 1; i++) {
        for (int j = 0; j <= (n + 1) / 2 - 1; j++) {
            int temp = matrix[i][j];
            matrix[i][j] = matrix[n - 1 - j][i];
            matrix[n - 1 - j][i] = matrix[n - 1 - i][n - 1 - j];
            matrix[n - 1 - i][n - 1 - j] = matrix[j][n - 1 - i];
            matrix[j][n - 1 - i] = temp;
        }
    }
}
```

解法3：用翻转代替旋转：可用 **一次水平翻转 + 一次转置变换 模拟顺时针旋转90°**

* 水平翻转：`(i, j) <-> (n - 1 - i, j)` 枚举矩阵上半部分 `i` `[0...n/2-1]`, `j` `[0...n-1]`
* 转置变换（主对角线翻转）：`(i, j) <-> (j, i)` 枚举对角线左下方 `i` `[0...n-1]`, `j` `[0...i-1]`
* 联合可得：`(i, j) <-> (n - 1 - i, j) <-> (j, n - 1 - i)` 与顺时针旋转 `90°` 对应公式相同
* 注意：虽然每次变换只枚举一半的元素，但变换公式是对任意的 `(i, j)` 都成立的，这是因为每次变换会让一对数双双到达正确的位置。因此可将矩阵的每个点都视作 `(i, j)`，每个点都经历了 `(i, j) -> (n - 1 - i, j) -> (j, n - 1 - i)` 的过程

复杂度：`O(n`^2^`)` `O(1)`

```java
public void rotate3(int[][] matrix) {
    int n = matrix.length;
    for (int i = 0; i <= n / 2 - 1; i++) {  // 水平翻转
        for (int j = 0; j <= n - 1; j++) {
            int temp = matrix[i][j];
            matrix[i][j] = matrix[n - 1 - i][j];
            matrix[n - 1 - i][j] = temp;
        }
    }
    for (int i = 0; i <= n - 1; i++) {  // 转置变换（主对角线翻转）
        for (int j = 0; j <= i - 1; j++) {
            int temp = matrix[i][j];
            matrix[i][j] = matrix[j][i];
            matrix[j][i] = temp;
        }
    }
}
```

## [所有子字符串中的元音](https://leetcode.cn/problems/vowels-of-all-substrings/)(2063)

问题：给你一个字符串 `word` ，返回 `word` 的所有子字符串中 **元音的总数** ，元音是指 `'a'`、`'e'`*、*`'i'`*、*`'o'` 和 `'u'` *。**子字符串** 是字符串中一个连续（非空）的字符序列。**注意：**由于对 `word` 长度的限制比较宽松，答案可能超过有符号 32 位整数的范围。

分析：计算每个元音（在所有包含它的子串）中的贡献：对于长度为 `n` 的字符串，包含第 `i` 位子串的数量

枚举子串的左端点，共有 `i + 1` 种选法；枚举子串的右端点，共有 `n - i` 种选法，共计 `(i + 1) * (n - i)` 种

```java
private boolean isVowel(char c) { // 判断一个字符是否为元音（考虑大小写）
    return "aeiouAEIOU".indexOf(c) != -1;
}

public long countVowels(String word) {
    long totalVowels = 0L;
    int n = word.length();
    for (int i = 0; i < n; i++) {
        char currentChar = word.charAt(i);
        if (isVowel(currentChar)) {
            long contribution = (long)(i + 1) * (n - i); // 每个元音字符的贡献为 (i + 1) * (n - i)
            totalVowels += contribution;
        }
    }
    return totalVowels;
}
```
