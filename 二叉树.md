## [二叉树的前序遍历](https://leetcode.cn/problems/binary-tree-preorder-traversal/)(144)

问题：给你二叉树的根节点 `root` ，返回它节点值的 **前序** 遍历。

分析：递归

* 函数定义：前序遍历二叉树，`res` 记录当前遍历的节点
* 边界条件：空节点直接返回
* 转移关系：先记录当前节点，再递归到左子树、右子树

复杂度：`O(n)` `O(n)`

```java
public List<Integer> preorderTraversal(TreeNode root) {
    List<Integer> res = new ArrayList<Integer>();
    preorder(root, res);
    return res;
}

public void preorder(TreeNode root, List<Integer> res) { // 前序遍历（递归）
    if (root == null) {
        return;
    }
    res.add(root.val);
    preorder(root.left, res);
    preorder(root.right, res);
}
```

解法2：利用栈模拟递归过程，先找到最左端节点，然后处理右子树

复杂度：`O(n)` `O(n)`

```java
public List<Integer> preorderTraversal(TreeNode root) {
    List<Integer> res = new ArrayList<Integer>();
    Deque<TreeNode> stack = new ArrayDeque<TreeNode>();
    // 若左子树处理完则栈为空，但此时右子树未处理，故需判断 root != null
    while (!stack.isEmpty() || root != null) {
        while (root != null) { // 找到最左端的节点
            res.add(root.val); // 左节点入栈前记录（根->左）
            stack.push(root); // 左节点不断入栈
            root = root.left;
        }
        root = stack.pop(); // 最左端节点出栈（并视为子树的根节点）
        root = root.right; // 处理该节点的右子树（保证了左->右的顺序）
    }
    return res;
}
```

## [二叉树的后序遍历](https://leetcode.cn/problems/binary-tree-postorder-traversal/)(145)

问题：给你一棵二叉树的根节点 `root` ，返回其节点值的 **后序遍历** 。

分析：递归

* 函数定义：后序遍历二叉树，`res` 记录当前遍历的节点
* 边界条件：空节点直接返回
* 转移关系：先递归到左子树、右子树，再记录当前节点

复杂度：`O(n)` `O(n)`

```java
public List<Integer> postorderTraversal(TreeNode root) {
    List<Integer> res = new ArrayList<Integer>();
    postorder(root, res);
    return res;
}

public void postorder(TreeNode root, List<Integer> res) {
    if (root == null) {
        return;
    }
    postorder(root.left, res);
    postorder(root.right, res);
    res.add(root.val);
}
```

解法2：利用栈模拟递归过程，先找到最左端节点，再找到最右端。若当前节点是上一个节点的左节点，尝试访问上个节点的右子树。

复杂度：`O(n)` `O(n)`

```java
public List<Integer> postorderTraversal(TreeNode root) {
    List<Integer> res = new ArrayList<>();
    Deque<TreeNode> stack = new LinkedList<>();
    // 若左子树处理完则栈为空，但此时右子树未处理，故需判断 root != null
    while(!stack.isEmpty() || root != null){
        // 每次将当前节点压入栈中，如果当前节点有左子树，就往左子树跑，没有左子树就往右子树跑
        while(root != null){ // 先找到最左端，再找到最右端，直至叶子节点（左->右）
            stack.push(root);
            if(root.left != null){ // 有左节点就一直走
                root = root.left;
            }else{ // 没有左节点，向右走一步在判断
                root = root.right;
            }
        }
        root = stack.pop(); // 弹出当前节点（叶子节点）
        res.add(root.val);  // 当前节点一定是按后序遍历的下一个节点
        if(!stack.isEmpty() && root == stack.peek().left){ // 如果当前节点是上一个节点的左节点
            root = stack.peek().right; // 尝试访问上个节点的右子树，以符合后序遍历的顺序
        }else{ // 如果当前节点是上一个节点的右节点，那当前栈的栈顶元素继续弹出（右->根）
            root = null; // 置为null，在下一个循环中不会再向左搜索子树
        }
    }
    return res;
}
```

## [二叉树的层序遍历](https://leetcode.cn/problems/cong-shang-dao-xia-da-yin-er-cha-shu-lcof/)(LCR149)

问题：一棵圣诞树记作根节点为 `root` 的二叉树，节点值为该位置装饰彩灯的颜色编号。请按照从 **左** 到 **右** 的顺序返回每一层彩灯编号。

分析：利用队列存储每一层节点，出队时将下一层的左右孩子入队

复杂度：`O(n)` `O(n)`

```java
public int[] decorateRecord(TreeNode root) {
    if(root == null){
        return new int[0];
    }
    List<Integer> res = new ArrayList<>();
    Queue<TreeNode> queue = new LinkedList<>();
    queue.add(root);
    while(!queue.isEmpty()){ // 依次将每一层节点入队、出队
        TreeNode node = queue.poll();
        res.add(node.val); // 记录当前出队节点
        if(node.left != null){ // 先将左孩子入队
            queue.add(node.left);
        }
        if(node.right != null){ // 再将右孩子入队
            queue.add(node.right);
        }
    }
    int ans[] = new int[res.size()];
    for(int i = 0; i < res.size(); i++){ // 将ArrayList转为int[]
        ans[i] = res.get(i);
    }
    return ans;
}
```

## [二叉树的逐层层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/)(102)

问题：给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

分析：利用队列存储每一层节点，出队时将下一层的左右孩子入队，同时用列表记录每一层的节点依次放入二维列表

复杂度：`O(n)` `O(n)`

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> ans = new ArrayList<>();  // 声明动态二维数组存储答案
    if (root == null) {
        return ans;
    }
    Queue<TreeNode> q = new ArrayDeque<>();  // 双端队列
    q.add(root);  // 根结点入队
    while (!q.isEmpty()) {  // 队列不为空时
        int n = q.size();  // 获取当前队列（当前层）的结点数量
        List<Integer> vals = new ArrayList<>(n); // 根据已知容量创建列表
        while (n-- > 0) {  // 将当前层全部出队，并将下一层全部入队
            TreeNode node = q.remove();  // 队头元素出队
            vals.add(node.val);  // 出队时访问
            if (node.left != null) {  // 将该结点的左右孩子入队（下一层）
                q.add(node.left);
            }
            if (node.right != null) {
                q.add(node.right);
            }
        }
        ans.add(vals);  // 将当前层元素加入答案中
    }
    return ans;
}
```

## [二叉树的锯齿形层序遍历](https://leetcode.cn/problems/binary-tree-zigzag-level-order-traversal/)(103)

问题：给你二叉树的根节点 `root` ，返回其节点值的 **锯齿形层序遍历** 。（即先从左往右，再从右往左进行下一层遍历，以此类推）。

分析：利用队列存储每一层节点，出队时将下一层的左右孩子入队，同时用列表记录每一层的节点依次放入二维列表。同时设置 `even` 记录层数的奇偶，将偶数层列表元素反转即可

复杂度：`O(n`^2^`)` `O(n)`

```java
public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
    List<List<Integer>> ans = new ArrayList<>();  // 声明动态二维数组存储答案
    if (root == null) {
        return ans;
    }
    Queue<TreeNode> q = new ArrayDeque<>();  // 双端队列
    q.add(root);  // 根结点入队
    for (boolean even = false; !q.isEmpty(); even = !even) {  // 加入奇偶标志，每次循环反转标志实现奇偶控制
        int n = q.size();  // 获取当前队列（当前层）的结点数量
        List<Integer> vals = new ArrayList<>(n); // 根据已知容量创建列表
        while (n-- > 0) {  // 将当前层全部出队，并将下一层全部入队
            TreeNode node = q.remove();  // 队头元素出队
            vals.add(node.val);  // 出队时访问
            if (node.left != null) {  // 将该结点的左右孩子入队（下一层）
                q.add(node.left);
            }
            if (node.right != null) {
                q.add(node.right);
            }
        }
        if (even) {  // 若当前为偶数层则将元素反转
            Collections.reverse(vals);
        }
        ans.add(vals);  // 将当前层元素加入答案中
    }
    return ans;
}
```

```java
public int findBottomLeftValue(TreeNode root) {
    TreeNode node = root;

            q.add(node.left);
```

## [二叉树的右视图](https://leetcode.cn/problems/binary-tree-right-side-view/)(199)

问题：给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

分析：逐层层序遍历，每层的右端节点即为右视图

复杂度：`O(n)` `O(n)`

```java
public List<Integer> rightSideView(TreeNode root) {
    List<Integer> ans = new LinkedList<>();  // 声明动态二维数组存储答案
    if (root == null) {

    q.add(root);  // 根结点入队

        List<Integer> vals = new ArrayList<>(n); // 根据已知容量创建列表

            TreeNode node = q.remove();  // 队头元素出队

            if (node.left != null) {  // 将该结点的左右孩子入队（下一层）
                q.add(node.left);
            }
            if (node.right != null) {
                q.add(node.right);
            }
        }
        ans.add(vals.get(vals.size() - 1));  // 返回层序遍历时每层的右节点即可
    }

```

解法2：递归

* 函数定义：遍历以 `root` 为根节点，深度为 `depth` 的子树
* 边界条件：空节点直接返回
  * 当 `depth == ansList.size()` 时，表明第一次到达当前深度，故计入答案中
* 转移关系：依次递归右子树、左子树（先遍历右子树保证得到右视图）

复杂度：`O(n)` `O(n)`

```java
class Solution {
    private final List<Integer> ansList = new ArrayList<>();  // 定义动态数组记录答案

    private void dfs2(TreeNode root, int depth) {
        if (root == null){
            return;
        }
        if (depth == ansList.size()){  // 当深度等于答案数组长度时，表明第一次到达当前深度，故计入答案中
            ansList.add(root.val);
        }
        dfs2(root.right, depth + 1);  // 由于是右视图，故先遍历右子树
        dfs2(root.left, depth + 1);
    }

    public List<Integer> rightSideView(TreeNode root) {
        dfs2(root, 0);
        return ansList;
    }
}
```

## [二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/)(104)

问题：给定一个二叉树 `root` ，返回其最大深度。二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

分析：递归（以方法一：不用全局变量，有返回值为例）

* 函数定义：对以 `root` 为根节点的子树，返回其最大深度
* 边界条件：空节点返回深度 `0`
* 转移关系：先递归到左子树、右子树，再计算并返回当前节点的最大深度 `Math.max(lDepth, rDepth) + 1`

复杂度：`O(n)` `O(n)`

```java
class Solution { // O(n) O(n)
    // 方法一：不用全局变量，有返回值
    public int maxDepth(TreeNode root) { // 表示以root为根节点的子树，返回其最大深度
        if (root == null) return 0;
        int lDepth = maxDepth(root.left);
        int rDepth = maxDepth(root.right);
        return Math.max(lDepth, rDepth) + 1;
    }
    
    // 方法二：用全局变量，无返回值
    private int ans;
    private void dfs(TreeNode node, int cnt) { // 表示递归到node时，计数了cnt次（root到node距离为cnt）
        if (node == null) return;
        ++cnt;
        ans = Math.max(ans, cnt);
        dfs(node.left, cnt);
        dfs(node.right, cnt);
    }
    public int maxDepth2(TreeNode root) {
        dfs(root, 0);
        return ans;
    }
}
```

## [二叉树最大宽度](https://leetcode.cn/problems/maximum-width-of-binary-tree/)(662)

问题：给你一棵二叉树的根节点 `root` ，返回树的 **最大宽度** 。树的 **最大宽度** 是所有层中最大的 **宽度** 。每一层的 **宽度** 被定义为该层最左和最右的非空节点（即，两个端点）之间的长度。将这个二叉树视作与满二叉树结构相同，这些 `null` 节点也计入长度。

分析：利用 `Queue` 层序遍历的同时存储节点的编号（按完全二叉树编号），每一层的首尾节点标号差 `+1` 即为宽度

复杂度：`O(n)` `O(n)`

```java
public int widthOfBinaryTree(TreeNode root) {
    int ans = 0;
    if (root == null) {
        return 0;
    }
    Queue<Pair<TreeNode, Integer>> q = new ArrayDeque<>(); // 辅助进行层序遍历（同时存储编号）
    q.add(new Pair<TreeNode, Integer>(root, 1));  // 根结点入队
    while (!q.isEmpty()) {  // 队列不为空时
        int n = q.size();  // 获取当前队列（当前层）的结点数量
        List<Pair<TreeNode, Integer>> vals = new ArrayList<>(n); // 根据已知容量创建列表
        while (n-- > 0) {  // 将当前层全部出队，并将下一层全部入队
            Pair<TreeNode, Integer> pair = q.remove();  // 队头元素出队
            vals.add(pair);  // 出队时访问
            TreeNode keyNode = pair.getKey();
            int value = pair.getValue();
            if (keyNode.left != null) {  // 将该结点的左右孩子入队（下一层）
                q.add(new Pair<TreeNode, Integer>(keyNode.left, value * 2));
            }
            if (keyNode.right != null) { // 将左右孩子按照满二叉树规则编号
                q.add(new Pair<TreeNode, Integer>(keyNode.right, value * 2 + 1));
            }
        }
        // 每层首尾两个节点的标号之差+1即为宽度
        int width = vals.get(vals.size() - 1).getValue() - vals.get(0).getValue() + 1;
        ans = Math.max(ans, width); // 更新答案为每层宽度的最大值
    }
    return ans;
}
```

## [二叉搜索树中第 K 小的元素](https://leetcode.cn/problems/kth-smallest-element-in-a-bst/)(230)

问题：给定一个二叉搜索树的根节点 `root` ，和一个整数 `k` ，请你设计一个算法查找其中第 `k` 小的元素（从 1 开始计数）。

分析：递归（中序遍历），中序遍历二叉搜索树结果单调不减，返回第 `k` 个即可

复杂度：`O(n)` `O(n)`

```java
public int kthSmallest(TreeNode root, int k) {
    List<Integer> res = new ArrayList<Integer>();
    inorder(root, res);
    return res.get(k - 1); // 中序遍历二叉搜索树结果单调不减
}

public void inorder(TreeNode root, List<Integer> res) { // 中序遍历
    if (root == null) {
        return;
    }
    inorder(root.left, res);
    res.add(root.val);
    inorder(root.right, res);
}
```

解法2：递归（计算节点数量），再由左子树<根节点<右子树判断第 `k` 小的元素位置，继续递归 `kthSmallest`

* 函数定义：返回以 `root` 为根节点的子树的节点数量
* 边界条件：空节点返回数量 `0`
* 转移关系：先递归到左子树、右子树，再计算并返回当前根节点的节点数量 `leftNodes + rightNodes + 1`

复杂度：`O(n)` `O(h)`

```java
public int kthSmallest(TreeNode root, int k) {
    int leftNodes = countNodes(root.left);
    if(leftNodes < k - 1){ // 第k小的元素在右子树中
        return kthSmallest(root.right, k - leftNodes - 1); // 减去左子树的节点(leftNodes)以及根节点(1)
    }else if(leftNodes == k - 1){ // 第k小的元素为根节点
        return root.val;
    }else{ // 第k小的元素在左子树中
        return kthSmallest(root.left, k); // 左子树<根节点<右子树，故仍然搜索第k小的节点
    }
}

public int countNodes(TreeNode root){ // 定义：返回以root为根节点的子树的节点数量
    if(root == null){
        return 0;
    }
    int leftNodes = countNodes(root.left); // 递归处理左右子树并接收返回值
    int rightNodes = countNodes(root.right);
    return leftNodes + rightNodes + 1; // 计算当前节点数量
}
```

## [路径总和](https://leetcode.cn/problems/path-sum/)(112)

问题：给你二叉树的根节点 `root` 和一个表示目标和的整数 `targetSum` 。判断该树中是否存在 **根节点到叶子节点** 的路径，这条路径上所有节点值相加等于目标和 `targetSum` 。如果存在，返回 `true` ；否则，返回 `false` 。

解法2：递归

* 函数定义：以 `root` 为根节点的子树是否存在和为 `targetSum` 的路径
* 边界条件：
  * 递归到空节点返回 `false`
  * 递归到叶子节点，判断叶子值与剩余目标值是否相等 `return root.val == targetSum`
* 转移关系：先递归到左子树、右子树，同时减少目标值 `targetSum - root.val` ，再用 `||` 连接左右子树的判断结果

复杂度：`O(n)` `O(h)`

```java
// 定义：以root为根节点的子树是否存在和为targetSum的路径
public boolean hasPathSum(TreeNode root, int targetSum) {
    if(root == null){ // 递归到空节点，直接返回false
        return false;
    }
    if(root.left == null && root.right == null){ // 递归到叶子节点，判断叶子值与剩余目标值是否相等
        return root.val == targetSum;
    }
    // 若左子树或右子树有目标路径，表明此节点存在目标路径
    return hasPathSum(root.left, targetSum - root.val) || hasPathSum(root.right, targetSum - root.val);
}
```

## [路径总和 II](https://leetcode.cn/problems/path-sum-ii/)(113)

问题：给你二叉树的根节点 `root` 和一个整数目标和 `targetSum` ，找出所有 **从根节点到叶子节点** 路径总和等于给定目标和的路径。

解法2：回溯

* 函数定义：以 `root` 为根节点的子树是否存在和为 `targetSum` 的路径
* 边界条件：递归到空节点，直接返回
  * 递归到叶子节点且 `targetSum == 0`，此次搜索满足题意，复制一份路径记录答案
* 转移关系：依次递归到左子树、右子树
  * 当前问题：记录当前节点路径 `path.add(root.val)`
  * 子问题：当前节点是否为叶子节点且 `targetSum == 0`
  * 下一个子问题：依次递归到左子树、右子树

复杂度：`O(n`^2^`)` `O(n)`

```java
class Solution { // O(n^2)
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    public List<List<Integer>> pathSum(TreeNode root, int targetSum) {
        dfs(root, targetSum);
        return res;
    }

    // 定义：以root为根节点的子树是否存在和为targetSum的路径（并在回溯中记录路径与答案）
    public void dfs(TreeNode root, int targetSum){
        if(root == null){ // 递归到空节点，直接返回
            return;
        }
        path.add(root.val); // #记录路径#
        targetSum -= root.val;
        if(root.left == null && root.right == null && targetSum == 0){ // 此次搜索满足题意
            res.add(new ArrayList<>(path));  // 必须要复制一份，否则path改变ans也改变
        }
        dfs(root.left, targetSum); // 递归搜索左子树
        dfs(root.right, targetSum); // 递归搜索右子树
        path.remove(path.size() - 1); // #恢复现场#
    }
}
```

## [从前序与中序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)(105)

问题：给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

分析：递归

* 函数定义：根据前序序列 `[preL, preR]` 和中序序列 `[inL, inR]` 构造二叉树，并返回根节点
* 边界条件：`preL > preR` 递归到空节点，返回 `null`
* 转移关系：根据先序的 `root` 划分为左右子树递归，并将返回的根节点分别连接到左右子树

复杂度：`O(n)` `O(h)`

```java
class Solution { // O(n) O(n)
    Map<Integer, Integer> getInorderIdx = new HashMap(); // 建立起中序遍历节点与数组下标的对应关系

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        for(int i = 0; i < inorder.length; i++){
            getInorderIdx.put(inorder[i], i);
        }
        return build(preorder, 0, preorder.length - 1, inorder, 0, inorder.length - 1);
    }

    // 定义：根据前序序列 [preL, preR] 和中序序列 [inL, inR] 构造二叉树，并返回根节点
    public TreeNode build(int[] pre, int preL, int preR, int[] in, int inL, int inR) {
        if (preL > preR) { // preL > preR || inL > inR
            return null;
        }
        TreeNode root = new TreeNode(pre[preL]);  // 先序的第一个结点为根结点
        int inIndexOfRoot = getInorderIdx.get(pre[preL]);  // 根据先序的根结点位置分成左右两部分递归
        int leftCount = inIndexOfRoot - inL;  // 中序序列左半部分的长度
        // 根据root划分为左右子树递归，并将返回的根节点分别连接到左右子树
        root.left = build(pre, preL + 1, preL + leftCount, in, inL, inIndexOfRoot - 1);
        root.right = build(pre, preL + leftCount + 1, preR, in, inIndexOfRoot + 1, inR);
        return root;
    }
}
```

## [从中序与后序遍历序列构造二叉树](https://leetcode.cn/problems/construct-binary-tree-from-inorder-and-postorder-traversal/)(106)

问题：给定两个整数数组 `inorder` 和 `postorder` ，其中 `inorder` 是二叉树的中序遍历， `postorder` 是同一棵树的后序遍历，请你构造并返回这颗 *二叉树* 。

分析：递归

* 函数定义：根据后序序列 `[postL, postR]` 和中序序列 `[inL, inR]` 构造二叉树，并返回根节点
* 边界条件：`postL > postR` 递归到空节点，返回 `null`
* 转移关系：根据后序的 `root` 划分为左右子树递归，并将返回的根节点分别连接到左右子树

复杂度：`O(n)` `O(h)`

```java
class Solution { // O(n) O(n)
    Map<Integer, Integer> getInorderIdx = new HashMap(); // 建立起中序遍历节点与数组下标的对应关系

    public TreeNode buildTree(int[] inorder, int[] postorder) {
        for(int i = 0; i < inorder.length; i++){
            getInorderIdx.put(inorder[i], i);
        }
        return build(postorder, 0, postorder.length - 1, inorder, 0, inorder.length - 1);
    }

    // 定义：根据后序序列 [postL, postR] 和中序序列 [inL, inR] 构造二叉树，并返回根节点
    public TreeNode build(int[] post, int postL, int postR, int[] in, int inL, int inR) {
        if (postL > postR) { // postL > postR || inL > inR
            return null;
        }
        TreeNode root = new TreeNode(post[postR]);  // 后序的最后一个结点为根结点
        int inIndexOfRoot = getInorderIdx.get(post[postR]);  // 根据后序的根结点位置分成左右两部分递归
        int leftCount = inIndexOfRoot - inL;  // 中序序列左半部分的长度
        // 根据root划分为左右子树递归，并将返回的根节点分别连接到左右子树
        root.left = build(post, postL, postL + leftCount - 1, in, inL, inIndexOfRoot - 1);
        root.right = build(post, postL + leftCount, postR - 1, in, inIndexOfRoot + 1, inR);
        return root;
    }
}
```

分析：递归

* 函数定义：返回以 `root` 为根节点的镜像二叉树

* 转移关系：递归右子树，赋给左节点；递归左子树，赋给右节点（注意需要先接受再赋值，否则会影响第二次递归）

```java
public TreeNode invertTree(TreeNode root) {  // 定义：返回以root为根节点的镜像二叉树
    if(root == null){ // root == null || (root.left == null && root.right == null)
        return root;
    }
    TreeNode newLeft = invertTree(root.right); // 递归右子树，赋给左节点
    TreeNode newRight = invertTree(root.left); // 递归左子树，赋给右节点
    root.left = newLeft;
    root.right = newRight;
    return root;
}
```

## [二叉树的序列化与反序列化](https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/)(297)

问题：序列化是将一个数据结构或者对象转换为连续的比特位的操作，进而可以将转换后的数据存储在一个文件或者内存中，同时也可以通过网络传输到另一个计算机环境，采取相反方式重构得到原数据。请设计一个算法来实现二叉树的序列化与反序列化。这里不限定你的序列 / 反序列化算法执行逻辑，你只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。

分析：

* 序列化：利用队列和层序遍历（补充非空节点的 `null` 节点）顺序拼接为字符串（`,` 分隔符）
* 反序列化：利用队列将接收的字符串数组按层序解码，依次构建左右节点（忽略 `null` 节点）并入队

复杂度：`O(n)` `O(n)`

```java
public class Codec {
    // Encodes a tree to a single string.
    public String serialize(TreeNode root) { // O(n) O(n)
        if(root == null){
            return "";
        }
        StringBuilder sb = new StringBuilder();
        Queue<TreeNode> queue = new LinkedList<>();
        queue.add(root);
        while(!queue.isEmpty()){
            TreeNode node = queue.poll();
            if(node != null){ // 将左右节点全部入队，尽管可能为null
                queue.offer(node.left);
                queue.offer(node.right);
                sb.append(node.val + ",");
            }else{ // null节点没有左右节点，但要将其拼接到sb中
                sb.append("null,");
            }
        }
        return sb.toString();
    }

    // Decodes your encoded data to tree.
    public TreeNode deserialize(String data) { // O(n) O(n)
        if(data == null || data.length() == 0){
            return null;
        }
        String[] str = data.split(",");
        Queue<TreeNode> queue = new LinkedList<>();
        TreeNode root = new TreeNode(Integer.parseInt(str[0]));
        queue.offer(root);
        int i = 1; // 指向序列化字符串的下标位置
        while(!queue.isEmpty()){
            TreeNode node = queue.poll(); // 出队作为当前根节点
            if(!str[i].equals("null")){ // 构建左节点（忽略null）
                TreeNode left = new TreeNode(Integer.parseInt(str[i]));
                node.left = left;
                queue.offer(left);
            }
            i++;
            if(!str[i].equals("null")){ // 构建右节点（忽略null）
                TreeNode right = new TreeNode(Integer.parseInt(str[i]));
                node.right = right;
                queue.offer(right);
            }
            i++;
        }
        return root;
    }
}
```

## [验证二叉搜索树](https://leetcode.cn/problems/validate-binary-search-tree/)(98)

问题：给你一个二叉树的根节点 `root` ，判断其是否是一个有效的二叉搜索树。

解法1：前序遍历：先判断，再递归（由根结点确定左右孩子是否有效）

* 函数定义：判断以 `node` 为根节点，值域为 `(left, right)` 的子树是否是二叉搜索树
* 边界条件：递归到空节点，返回 `true`
* 转移关系：若当前节点值 `x` 在 `(left, right)` 内，且左子树值域为 `(left, x)`，右子树值域为`(x, right)`，则是二叉搜索树

复杂度：`O(n)` `O(n)/O(h)`

```java
private boolean isValidBST1(TreeNode node, long left, long right) {  // 重载方法
    if (node == null) {
        return true;
    }
    long x = node.val;
    return left < x && x < right && isValidBST1(node.left, left, x) && isValidBST1(node.right, x, right);
}

public boolean isValidBST1(TreeNode root) {
        return isValidBST1(root, Long.MIN_VALUE, Long.MAX_VALUE); // 整棵树的值域可以为 (-∞, +∞)
    }
```

解法2：中序遍历：大于上一个节点（由上一个结点确定下一个结点是否有效），用全局变量 `pre` 记录上一个节点值

* 函数定义：判断以 `node` 为根节点的子树是否是二叉搜索树
* 边界条件：递归到空节点，返回 `true`
* 转移关系：先递归判断左子树为 `lValue`，再比较当前节点值和 `pre` 的大小，若 `root.val <= pre` 则返回 `false`，接着更新 `pre` 值，最后递归判断右子树为 `rValue`（中序遍历的顺序），并返回 `lValue && rValue`
  * 优化：若左子树 `!lValue` 提前返回 `false`，最后将 `lValue && rValue` 简化为 `rValue`

复杂度：`O(n)` `O(n)/O(h)`

```java
private long pre = Long.MIN_VALUE; // 初始化为最小值，第一次比较时，任何数一定大于最小值（满足）

public boolean isValidBST2(TreeNode root) {
    if (root == null) {
        return true;
    }
    boolean lValue = isValidBST2(root.left);
    if (!lValue || root.val <= pre) {  // !lValue可以提前一路返回false，不用递归右子树
        return false;
    }
    pre = root.val;  // 保存当前结点值，供下一个结点比较
    boolean rValue = isValidBST2(root.right);
    //  return lValue && rValue;
    return rValue;  // 由于 lValue 已经提前判断过，相当于直接返回 rValue
}
```

解法3：后序遍历：先递归，再判断（由左右孩子确定双亲结点是否有效）

* 函数定义：返回以 `node` 为根节点的子树的 `[最小值, 最大值]`
* 边界条件：递归到空节点时，最小值初始化为 `Long.MAX_VALUE`，最大值初始化为 `Long.MIN_VALUE`（保证后续被更新）
* 转移关系：本方法在**返回时取最值**是关键所在，这不仅**统一了空节点时的判断**，还使得**一旦判断无效则一直返回不合法值**
  * 递归左子树，若当前节点值小于左子树最大值 `x <= left[1]`，则不是二叉搜索树，返回 `[Long.MIN_VALUE, Long.MAX_VALUE]`，即表明子树的最小值与最大值不合法，也保证了后续取最值时始终不合法
  * 递归右子树，若当前节点值大于右子树最小值 `x >= right[0]`，则不是二叉搜索树，返回 `[Long.MIN_VALUE, Long.MAX_VALUE]`，即表明子树的最小值与最大值不合法，也保证了后续取最值时始终不合法
  * 否则满足二叉搜索树的规则，返回当前根节点子树的**最小值 `Math.min(left[0], x)` 和最大值 `Math.max(right[1], x)`**，这里取最值是因为从空节点的 `Long.MAX_VALUE` 和 `Long.MIN_VALUE` 递归到父结点需要

复杂度：`O(n)` `O(n)/O(h)`

```java
private long[] dfs(TreeNode node) {
    if (node == null){
        return new long[]{Long.MAX_VALUE, Long.MIN_VALUE};  // 这样初始化在return时无穷必然被更新为有限值
    }   
    long x = node.val;
    long[] left = dfs(node.left);
    if (x <= left[1]){  // 在递归完左子树之后立刻判断，如果不是二叉搜索树，就不用递归右子树了
        return new long[]{Long.MIN_VALUE, Long.MAX_VALUE};  // 一旦这样赋值后会一直如此返回直至根结点
    }
    long[] right = dfs(node.right);
    if (x >= right[0]){  // 右边判断不成立
        return new long[]{Long.MIN_VALUE, Long.MAX_VALUE};
    }
    return new long[]{Math.min(left[0], x), Math.max(right[1], x)};  // 当前结点有效
}

public boolean isValidBST(TreeNode root) {
    // return dfs(root)[0] != Long.MIN_VALUE;
    return dfs(root)[1] != Long.MAX_VALUE; // 如果以root为根节点子树的最大值是无穷大，表明不是二叉搜索树
}
```

## [二叉搜索树的最近公共祖先-LCA](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-search-tree/)(235)

问题：给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。

分析：递归，由于二叉搜索树的特性，仅需比较 `p` 和 `q` 的值与 `root.val` 的大小即可判断在哪棵子树

* 函数定义：对以 `root` 为根节点的子树，返回 `p` 和 `q` 的最近公共祖先
* 边界条件：返回当前结点
  *  `p` 和 `q` 分别在左右子树
  * 当前节点是 `p`
  * 当前节点是 `q`

* 转移关系：
  *  `p` 和 `q` 都在左子树：返回递归左子树的结果
  *  `p` 和 `q` 都在左子树：返回递归右子树的结果


复杂度：`O(h)` `O(h)`

```java
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    int x = root.val;  // 当前节点一定不是空节点，无需判断
    if (p.val < x && q.val < x) {  // 1. p q 都在左子树，返回递归左子树的结果
        return lowestCommonAncestor(root.left, p, q);
    } else if (p.val > x && q.val > x) {  // 2. p q 都在右子树，返回递归右子树的结果
        return lowestCommonAncestor(root.right, p, q);
    } else {
        return root;  // 3. p q分别在左右子树 4.当前结点是 p 5.当前结点是 q  均返回当前结点
    }
}
```

## [二叉树的最近公共祖先-LCA](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/)(236)

问题：给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

分析：递归，由于不能判断 `p` 和 `q` 在哪棵子树，故每次都需递归左右子树

* 函数定义：对以 `root` 为根节点的子树，返回 `p` 和 `q` 的最近公共祖先，**否则返回所包含的 `p` 或 `q` 单个节点**
* 边界条件：返回当前节点
  * 当前节点为空
  * 当前节点是 `p `（会一直向上返回）
  * 当前节点是 `q `（会一直向上返回）

* 转移关系：先递归到左右子树，再判断
  * 左右子树均不为空：表明 `p` 和 `q` 分布在 `root` 的左右子树，返回当前节点作为最近公共祖先
  * 左右子树有一个不为空：表明 `root` 的子树仅有单个节点 `p` 或 `q` ，返回该节点以便后续判断最近公共祖先
  * 左右子树均为空：表明 `root` 的子树无目标节点，返回 `null`（可合并到上一种情况）


复杂度：`O(n)` `O(h)`

```java
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q)  // 类似先序遍历，因为一开始先比较了 root 是否等于 p 或 q
        return root;
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    if (left != null && right != null)
        return root;
    return left != null ? left : right;
}
```

# 平衡二叉树

## [平衡二叉树](https://leetcode.cn/problems/balanced-binary-tree/)(110)

问题：给定一个二叉树，判断它是否是平衡二叉树（一个二叉树*每个节点* 的左右两个子树的**高度差的绝对值不超过 1** )。

分析：递归

* 函数定义：对以 `root` 为根节点的子树，返回其最大深度
* 边界条件：空节点返回深度 `0`
* 转移关系：先递归到左子树、右子树，再计算并返回当前节点的最大深度 `Math.max(lDepth, rDepth) + 1`

复杂度：`O(n)` `O(h)`

```java
private int getHeight(TreeNode node) {
    if (node == null){  // base case: 空结点高度为0
        return 0;
    }
    int leftH = getHeight(node.left);
    if (leftH == -1){  // 如果收到-1，一路返回-1退出递归
        return -1;
    }
    int rightH = getHeight(node.right);
    if (rightH == -1 || Math.abs(leftH - rightH) > 1){  // 第一个-1通过判断不平衡返回
        return -1;
    }
    return Math.max(leftH, rightH) + 1;  // 平衡状态下返回子树高度
}

public boolean isBalanced(TreeNode root) {
    return getHeight(root) != -1;
}
```
