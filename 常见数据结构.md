# 栈（单调栈）

## 反转栈(仅递归)

问题：给你一个栈 `stack`，需要把里面的元素进行反转。不能使用数组、队列、栈等数据结构辅助，仅能使用递归。

解法1：构造 `Stack<Integer> pushLast(Stack<Integer> stack, int x)` 实现将元素 `x` 插入 `stack` 的栈底，并返回 `stack`

```java
// 函数定义：反转栈 stack 里的元素，并返回 stack
public static Stack<Integer> reverse(Stack<Integer> stack){ // stack: [1, 2, 3, 4] -> [4, 3, 2, 1]
    // 边界条件：栈为空或只有一个元素，直接返回栈本身
    if(stack.isEmpty() || stack.size() == 1){
        return stack;
    }
    // 转移关系：通过弹出 top 减小递归规模，还需定义辅助函数 pushLast 将弹出的 top 插入栈底
    int top = stack.pop(); // top = 4 , stack = [1, 2, 3]
    stack = reverse(stack); // stack = [3, 2, 1]
    stack = pushLast(stack, top); // stack = [4, 3, 2, 1]
    return stack;
}

// 函数定义：将元素 x 插入 stack 的栈底，并返回 stack
public static Stack<Integer> pushLast(Stack<Integer> stack, int x){ // stack = [3, 2, 1], x = 4
    // 边界条件：栈为空，直接插入 x 并返回 stack
    if(stack.isEmpty()){
        stack.push(x);
        return stack;
    }
    // 转移关系：通过弹出 top 减小递归规模，递归调用栈底插入 x 后，再将 top 入栈
    int top = stack.pop(); // top = 1, stack = [3, 2]
    stack = pushLast(stack, x); // stack = [4, 3, 2]
    stack.push(top); // stack = [4, 3, 2, 1]
    return stack;
}
```

解法2：构造 `int popLast(Stack<Integer> stack)` 实现弹出 `stack` 的栈底元素 `last` 并返回

```java
// 函数定义：反转栈 stack 里的元素，并返回 stack
public static Stack<Integer> reverse(Stack<Integer> stack){ // stack: [1, 2, 3, 4] -> [4, 3, 2, 1]
    // 边界条件：栈为空或只有一个元素，直接返回栈本身
    if(stack.isEmpty() || stack.size() == 1){
        return stack;
    }
    // 转移关系：通过取栈底元素 last:popLast() 减小递归规模，最后将 last 入栈
    int last = popLast(stack); // last = 1 , stack = [2, 3, 4]
    stack = reverse(stack); // stack = [4, 3, 2]
    stack.push(last); // stack = [4, 3, 2, 1]
    return stack;
}

// 函数定义：弹出 stack 的栈底元素 last 并返回
public static int popLast(Stack<Integer> stack){ // stack = [1, 2, 3, 4]
    // 边界条件：栈中仅有一个元素，直接弹出 last 并返回
    if(stack.size() == 1){
        return stack.pop();
    }
    // 转移关系：通过弹出 top 减小递归规模，递归调用弹出栈底 last 后，再将 top 入栈
    int top = stack.pop(); // top = 4, stack = [1, 2, 3]
    int last = popLast(stack); // last = 1, stack = [2, 3]
    stack.push(top); // stack = [2, 3, 4]
    return last;
}
```

## [用栈实现队列](https://leetcode.cn/problems/implement-queue-using-stacks/)(232)

问题：请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（`push`、`pop`、`peek`、`empty`）

分析：仅输入到 `inStack`，仅从 `outStack` 中输出（为空则先把 `inStack` 中所有元素转移到 `outStack`）

```java
class MyQueue {
    Deque<Integer> inStack, outStack; // Stack<Integer>，双端队列Deque可以同时模拟队列和栈

    public MyQueue() {
        inStack = new ArrayDeque<Integer>(); // Stack<Integer>
        outStack = new ArrayDeque<Integer>();
    }
    
    public void push(int x) { // 仅输入到inStack
        inStack.push(x);
    }
    
    public int pop() { // 仅从outStack中输出
        if(outStack.isEmpty()){ // 如果outStack为空，先把inStack中所有元素转移到outStack
            while(!inStack.isEmpty()){
                outStack.push(inStack.pop());
            }
        }
        return outStack.pop();
    }
    
    public int peek() { // 同pop()
        if(outStack.isEmpty()){
            while(!inStack.isEmpty()){
                outStack.push(inStack.pop());
            }
        }
        return outStack.peek();
    }
    
    public boolean empty() { // inStack 与 outStack 均为空，表明无元素
        return inStack.isEmpty() && outStack.isEmpty();
    }
}
```

## [有效的括号](https://leetcode.cn/problems/valid-parentheses/)(20)

问题：给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

分析：利用哈希表定义匹配括号对，遇左括号入栈，遇右括号检查栈顶元素是否与之匹配，若匹配则出栈

复杂度：`O(n)` `O(n)`

```java
public boolean isValid(String s) {
    int n = s.length();
    if (n % 2 == 1) { // 奇数长度一定不满足
        return false;
    }
    Map<Character, Character> pairs = new HashMap<Character, Character>(); // 定义匹配对
    pairs.put('(', ')');
    pairs.put('[', ']');
    pairs.put('{', '}');
    Stack<Character> stack = new Stack<>(); // 利用栈按序匹配
    for (Character ch : s.toCharArray()) {
        if (pairs.containsKey(ch)) { // 左括号入栈
            stack.push(ch);
        } else { // 右括号，若和当前栈顶不匹配则不满足（注意调用stack.peek()需要先保证栈不为空）
            if (stack.isEmpty() || pairs.get(stack.peek()) != ch) {
                return false;
            }
            stack.pop(); // 若存在匹配对，则相应的左括号出栈
        }
    }
    return stack.isEmpty(); // 若栈为空，表明所有括号得到了正确的匹配
}
```

## [逆波兰表达式求值](https://leetcode.cn/problems/evaluate-reverse-polish-notation/)(150)

问题：给你一个字符串数组 `tokens` ，表示一个根据 [逆波兰表示法](https://baike.baidu.com/item/逆波兰式/128437) 表示的算术表达式。请你计算并返回一个表示表达式值的整数。

分析：遇数字入栈，遇操作符取出栈顶的两个操作数计算，并将计算值入栈

复杂度：`O(n)` `O(n)`

```java
public int evalRPN(String[] tokens) {
    Deque<Integer> stack = new ArrayDeque<Integer>();
    int n = tokens.length;
    for (int i = 0; i < n; i++) {
        String token = tokens[i];
        if (isNumber(token)) { // 若为数字，则入栈
            stack.push(Integer.parseInt(token)); // 将String转为Integer(int)
        } else { // 若为符号，取出栈顶的两个操作数计算，并将计算值入栈
            int num2 = stack.pop();
            int num1 = stack.pop();
            switch (token) { // 匹配计算符号
                case "+":
                    stack.push(num1 + num2);
                    break;
                case "-":
                    stack.push(num1 - num2);
                    break;
                case "*":
                    stack.push(num1 * num2);
                    break;
                case "/":
                    stack.push(num1 / num2);
                    break;
                default:
            }
        }
    }
    return stack.pop(); // 栈中剩余的唯一元素即为计算结果
}

public boolean isNumber(String token) { // 不等于这4种操作符即为数字
    return !("+".equals(token) || "-".equals(token) || "*".equals(token) || "/".equals(token));
}
```

## [最小栈](https://leetcode.cn/problems/min-stack/)(155)

问题：设计一个支持 `push` ，`pop` ，`top` 操作，并能在常数时间内检索到最小元素的栈。`int getMin()` 获取堆栈中的最小元素。

分析：因为栈底元素始终最后出栈，因此栈中的最小值<=栈底元素，且栈具有顺序性，每次只需记录<=当前最小值的元素

复杂度：`O(1)` `O(n)`

```java
class MinStack { // O(1) O(n)
    Deque<Integer> stack, minStack; // 辅助栈记录当前栈中的最小元素（一一对应）
    public MinStack() {
        stack = new ArrayDeque<Integer>();
        minStack = new ArrayDeque<Integer>();
        minStack.push(Integer.MAX_VALUE); // 存入最大值，方便后续每次入栈时取最小
    }
    
    public void push(int val) {
        stack.push(val);
        minStack.push(Math.min(val, minStack.peek())); // 每次取当前栈的最小值和当前元素的 最小值
    }
    
    public void pop() { // 比较 stack1.peek().intValue() == stack2.peek().intValue() 要比较值而不是地址
        stack.pop();
        minStack.pop();
    }
    
    public int top() {
        return stack.peek();
    }
    
    public int getMin() {
        return minStack.peek();
    }
}
```

## [商品折扣后的最终价格](https://leetcode.cn/problems/final-prices-with-a-special-discount-in-a-shop/)(1475)

问题：给你一个数组 `prices` ，其中 `prices[i]` 是商店里第 `i` 件商品的价格。商店里正在进行促销活动，如果你要买第 `i` 件商品，那么你可以得到与 `prices[j]` 相等的折扣，其中 `j` 是满足 `j > i` 且 `prices[j] <= prices[i]` 的 **最小下标** ，如果没有满足条件的 `j` ，你将没有任何折扣。请你返回一个数组，数组中第 `i` 个元素是折扣后你购买商品 `i` 最终需要支付的价格。

分析：倒序遍历 `prices`，并用单调栈中维护当前位置右边的更小的元素列表，**从栈底到栈顶（从右到左）的元素是单调递增的**。

* 每次移动到数组中一个新的位置 `i`，就将单调栈中所有 `> prices[i]` 的元素弹出单调栈
* 从而位置 `i`右边的第一个 `<= prices[i]` 的元素即为栈顶元素，如果栈为空则说明当前位置右边没有更大的元素
* 随后将 `prices[i]` 入栈，满足了单调性

复杂度：`O(n)` `O(n)`

```java
public int[] finalPrices(int[] prices) {
    Deque<Integer> stack = new ArrayDeque<>(); // 单调（递增）栈
    int n = prices.length;
    int res[] = new int[n];
    for(int i = n - 1; i >= 0; i--){ // 题意需与后续比较，故倒叙遍历
        while(!stack.isEmpty() && stack.peek() > prices[i]){ // 大于目标值无法折扣
            stack.pop();
        }
        res[i] = stack.isEmpty() ? prices[i] : prices[i] - stack.peek(); // 栈不空则可折扣
        stack.push(prices[i]); // 更新栈，且由于出栈了更大的元素，一定满足单调性
    }
    return res;
}
```

## [每日温度](https://leetcode.cn/problems/daily-temperatures/)(739)

问题：给定一个整数数组 `temperatures` ，表示每天的温度，返回一个数组 `answer` ，其中 `answer[i]` 是指对于第 `i` 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 `0` 来代替。

分析：**及时去掉无用数据，保证双端队列有序**

解法1：从右到左：栈中记录下一个更大元素的「候选项」。

复杂度：`O(n)` `O(min(n, U))`，其中 `U = max(temperatures) − min(temperatures) + 1`。

```java
public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] ans = new int[n];
    Deque<Integer> st = new ArrayDeque<>();
    for (int i = n - 1; i >= 0; i--) { // 从右往左遍历
        int t = temperatures[i];
        // 将栈中不超过t的无用数据弹出，因为已不可能成为下一个更大值
        while (!st.isEmpty() && t >= temperatures[st.peek()]) {
            st.pop();
        }
        if (!st.isEmpty()) { // 记录答案，当前栈顶元素为t的下一个更大值
            ans[i] = st.peek() - i;
        }
        st.push(i); // 将当前元素t的下标i入栈
    }
    return ans;
}
```

解法2：从左到右：栈中记录还没算出「下一个更大元素」的那些数（的下标）。

复杂度：`O(n)` `O(n)`

```java
public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] ans = new int[n];
    Deque<Integer> st = new ArrayDeque<>();
    for (int i = 0; i < n; i++) { // 从左到右遍历
        int t = temperatures[i];
        // 将栈中小于t的元素出栈，因为t即这些元素的下一个更大值
        while (!st.isEmpty() && t > temperatures[st.peek()]) {
            int j = st.pop();
            ans[j] = i - j; // 依次出栈并计算与t的下标差，记录答案
        }
        st.push(i); // 将当前元素t的下标i入栈
    }
    return ans;
}
```

## [132模式](https://leetcode.cn/problems/132-pattern/)(456)

问题：给你一个整数数组 `nums` ，数组中共有 `n` 个整数。**132 模式的子序列** 由三个整数 `nums[i]`、`nums[j]` 和 `nums[k]` 组成，并同时满足：`i < j < k` 和 `nums[i] < nums[k] < nums[j]` 。如果 `nums` 中存在 **132 模式的子序列** ，返回 `true` ；否则，返回 `false` 。

分析：枚举 `j`，并使用合适的数据结构维护 `i` 和 `k`，使得满足132模式的可能性最大

* 由于 `1` 是模式中的最小值，因此**维护左侧元素 `nums[0...j−1]` 的最小值**即可，且满足 `nums[i] < nums[j]`
* 由于 `2` 是模式中的次小值，因此可以使用一个**有序集合（例如平衡树）维护右侧元素 `a[j+1...n−1]` 中的所有值**。当确定了 `nums[i]` 和 `nums[j]` 后，只需**在有序集合中查询严格比 `nums[i]` 大的那个最小的 `nums[k]`**，且满足 `nums[k] < nums[j]`

```java
public boolean find132pattern(int[] nums) {
    int n = nums.length;
    if (n < 3) {
        return false;
    }
    int leftMin = nums[0]; // 左侧最小值
    TreeMap<Integer, Integer> rightAll = new TreeMap<Integer, Integer>(); // 右侧所有元素
    for (int k = 2; k < n; k++) { // 统计右侧所有元素出现次数
        rightAll.put(nums[k], rightAll.getOrDefault(nums[k], 0) + 1);
    }
    for (int j = 1; j < n - 1; j++) { // 枚举 j
        if (leftMin < nums[j]) { // 如果存在 i
            Integer next = rightAll.ceilingKey(leftMin + 1); // 查询 k
            if (next != null && next < nums[j]) {
                return true;
            }
        }
        leftMin = Math.min(leftMin, nums[j]); // 维护左侧最小值
        rightAll.put(nums[j + 1], rightAll.get(nums[j + 1]) - 1); // 维护右侧
        if (rightAll.get(nums[j + 1]) == 0) {
            rightAll.remove(nums[j + 1]);
        }
    }
    return false;
}
```

解法2：枚举 `i`，**从后往前维护一个单调递减的栈，同时使用 `kMax` 记录所有出栈元素的最大值**（代表结构 `2`）。

* 通过维护「单调递减」确保已经找到了有效的 `(j, k)`，即 **`kMax` 的值是因为维护单调栈而被弹出的**，一定有 `nums[j] > kMax`
* 遍历到 `i`，只要发现存在 `nums[i] < kMax`，说明找到了有效的 `(i, k)`

```java
public boolean find132pattern(int[] nums) {
    int n = nums.length;
    Deque<Integer> d = new ArrayDeque<>();
    int kMax = Integer.MIN_VALUE;
    for (int i = n - 1; i >= 0; i--) { // 枚举 i
        if (nums[i] < kMax) { // 找到了有效的 (i, k)
            return true;
        }
        while (!d.isEmpty() && d.peekLast() < nums[i]) { // 维护单调栈的过程中找到了有效的 (j, k)
            kMax = Math.max(kMax, d.pollLast()); // 事实上，kMax 的变化也具有单调性，可写为 kMax = pollLast();
        }
        d.addLast(nums[i]);
    }
    return false;
}
```

## [用队列实现栈](https://leetcode.cn/problems/implement-stack-using-queues/)(225)

问题：请你仅使用两个队列实现一个后入先出（LIFO）的栈，并支持普通栈的全部四种操作（`push`、`top`、`pop` 和 `empty`）。

分析：先将新元素入队，再将队中原有元素依次出队再入队，使得新元素位于队头

```java
class MyStack {
    Queue<Integer> queue;

    public MyStack() {
        queue = new LinkedList<Integer>();
    }
    
    public void push(int x) {
        int n = queue.size(); // 获取队中原有元素的数量
        queue.offer(x); // 先把新元素入队
        for(int i = 0; i < n; i++){
            queue.offer(queue.poll()); // 队中原有元素出队再入队，使得新元素位于队头
        }
    }
    
    public int pop() {
        return queue.poll(); // 出队
    }
    
    public int top() {
        return queue.peek();
    }
    
    public boolean empty() {
        return queue.isEmpty();
    }
}
```

## [滑动窗口最大值](https://leetcode.cn/problems/sliding-window-maximum/)(239)

问题：给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。返回 *滑动窗口中的最大值* 。

分析：利用双端队列存储当前滑动窗口内的元素，**越早入队的应越大（否则会被更大且生命周期更长的上位替代）**，故单调递减。

* 若当前元素可以上位替代，从**队尾出队**更小的，再从**队尾入队**当前元素；若不可上位替代则直接**队尾入队**
* 若滑动窗口范围超出当前最大值，则将最先存入的**队头元素出队**
  * 由于需要判断最早的下标是否已经超出窗口范围，故将双端队列由存储元素转为**存储下标**
* 由于队头存放当前窗口的最大值，记录答案时仅需**查询队头元素**即可

总结：**及时去掉无用数据，保证双端队列有序**

复杂度：`O(n)` `O(min(k, U))`，其中 `U` 是 `nums` 中的不同元素个数（本题至多为 `20001`）

```java
public int[] maxSlidingWindow(int[] nums, int k) {
    // 双端列表（单调递减）存放下标，因为滑动窗口需要比较最早的下标是否已经超出窗口范围
    Deque<Integer> deque = new ArrayDeque<>();
    int n = nums.length, idx = 0;
    int res[] = new int[n - k + 1];
    for(int i = 0; i < n; i++){
        // 从队尾逐个比较，如果队尾<=当前元素，在以后的窗口中会被当前元素上位替代（更大且生命周期更长）
        while(!deque.isEmpty() && nums[deque.peekLast()] <= nums[i]){
            deque.pollLast(); // 从队尾出队被上位替代的元素
        }
        deque.offerLast(i); // 从队尾加入当前元素，更前端的元素虽然更大，但是生命周期更短
        if(i - deque.peekFirst() > k - 1){ // 滑动窗口范围超出当前最大值
            deque.pollFirst(); // 将最先存入的队头元素出队（尽管其最大）
        }
        if(i >= k - 1){ // 从第一个滑动窗口位置开始记录答案
            res[idx++] = nums[deque.peekFirst()]; // 队头存放当前窗口的最大值
        }
    }
    return res;
}
```

# 优先队列（堆）

## [数组中的第K个最大元素](https://leetcode.cn/problems/kth-largest-element-in-an-array/)(215)

问题：给定整数数组 `nums` 和整数 `k`，请返回数组中第 `k` 个最大的元素。

分析：利用优先队列，维护大小为 `k` 的小顶堆，元素>=堆顶则入队。遍历后获得前 `k` 个最大的元素，且堆顶最小恰为第 `k` 个

复杂度：`O(nlogk)` `O(k)`

```java
public int findKthLargest(int[] nums, int k) {
    Queue<Integer> minQueue = new PriorityQueue<Integer>(); // 默认是小顶堆
    for(int i = 0; i < nums.length; i++){
        if(minQueue.size() < k){ // 堆不足k时直接进队
            minQueue.offer(nums[i]);
        }else if(nums[i] > minQueue.peek()){ // 容量达到k且有更大的元素
            minQueue.poll(); // 出队最小的元素
            minQueue.offer(nums[i]); // 将当前元素入队
        }
    }
    return minQueue.peek(); // 前k大里的最小值即位第k大的元素
}
```

## [前 K 个高频元素](https://leetcode.cn/problems/top-k-frequent-elements/)(347)

问题：给你一个整数数组 `nums` 和一个整数 `k` ，请你返回其中出现频率前 `k` 高的元素。你可以按 **任意顺序** 返回答案。

分析：先利用哈希表统计每个元素出现的次数，再利用优先队列存储频率前 `k` 高的元素（按频率排序的小顶堆）

复杂度：`O(nlogk)` `O(n)`

```java
public int[] topKFrequent(int[] nums, int k) {
    // 使用字典（哈希表），统计每个元素出现的次数，元素为键，元素出现的次数为值
    HashMap<Integer,Integer> map = new HashMap();
    for(int num : nums){
        map.put(num, map.getOrDefault(num, 0) + 1); // 使用getOrDefault避免返回null
    }
    // 遍历map，用最小堆保存频率最大的k个元素
    PriorityQueue<Integer> pq = new PriorityQueue<>(new Comparator<Integer>() {
        @Override
        public int compare(Integer a, Integer b) { // 按频率排序的小顶堆
            return map.get(a) - map.get(b);
        }
    });
    for (Integer key : map.keySet()) {
        if (pq.size() < k) { // 堆不足k时直接进队
            pq.add(key);
        } else if (map.get(key) > map.get(pq.peek())) { // 容量达到k且有更大的元素
            pq.remove(); // 出队最小的元素
            pq.add(key); // 将当前元素入队
        }
    }
    int res[] = new int[k];
    for(int i = 0; i < k; i++){ // 取出最小堆中的元素（前k大）
        res[i] = pq.poll();
    }
    return res;
}
```

## [数据流的中位数](https://leetcode.cn/problems/find-median-from-data-stream/)(295)

问题：**中位数**是有序整数列表中的中间值。如果列表的大小是偶数，则没有中间值，中位数是两个中间值的平均值。

分析：保持不变量：len(low) == len(high) 或 len(low) == len(high) + 1；**max(low) <= min(high)**

* 维护存储较小部分的大顶堆，若两个堆大小相等，先存入小顶堆，再转入大顶堆
* 维护存储较大部分的小顶堆，若大顶堆比小顶堆多一个，先存入大顶堆，再转入小顶堆

```java
class MedianFinder {
    Queue<Integer> lowMaxQue; // 存储较小部分的大顶堆（若为奇数将多余的中位数存于此）
    Queue<Integer> highMinQue; // 存储较大部分的小顶堆

    public MedianFinder() {
        lowMaxQue = new PriorityQueue<Integer>((a, b) -> (b - a)); // 改为大顶堆
        highMinQue = new PriorityQueue<Integer>((a, b) -> (a - b)); // 默认为小顶堆
    }
    
    public void addNum(int num) { // O(logn)
        if(lowMaxQue.size() == highMinQue.size()){ // 偶数，即两个堆大小相等，需存入大顶堆中
            highMinQue.offer(num); // 先存入小顶堆
            lowMaxQue.offer(highMinQue.poll()); // 再转入大顶堆（利用自动排序，转入的一定是最小的）
        }else{ // 奇数，即大顶堆比小顶堆多一个，需存入小顶堆中
            lowMaxQue.offer(num); // 先存入大顶堆
            highMinQue.offer(lowMaxQue.poll()); // 再转入小顶堆（利用自动排序，转入的一定是最大的）
        }
    }
    
    public double findMedian() { // O(1)
        if(lowMaxQue.size() == highMinQue.size() + 1){ // 奇数，答案存储在大顶堆中
            return lowMaxQue.peek(); // * 1.0
        }else{ // 偶数，即返回两个堆顶元素的平均值
            return (lowMaxQue.peek() + highMinQue.peek()) / 2.0;
        }
    }
}
```

问题：展览馆展出来自 13 个朝代的文物，每排展柜展出 5 个文物。某排文物的摆放情况记录于数组 `places`，其中 `places[i]` 表示处于第 `i` 位文物的所属朝代编号。其中，编号为 0 的朝代表示未知朝代。请判断并返回这排文物的所属朝代编号是否能够视为连续的五个朝代（如遇未知朝代可算作连续情况）。

分析：设置集合检查重复元素，若有重复则一定不连续，若无重复且最值相差小于 `5` 则连续

复杂度：`O(n)` `O(n)`

```java
public boolean checkDynasty(int[] nums) {
    Set<Integer> set = new HashSet<>(); // 设置集合检查重复元素
    int min = 13, max = 0;
    for(int i = 0; i < nums.length; i++){
        if(nums[i] == 0){
            continue;
        }
        if(set.contains(nums[i])){ // 集合已有则重复，一定不连续
            return false;
        }
        set.add(nums[i]);
        max = Math.max(max, nums[i]);
        min = Math.min(min, nums[i]);
    }
    return max - min < 5; // 不重复，最值相差小于5则连续
}
```

## [两数之和](https://leetcode.cn/problems/two-sum/)(1)

问题：给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** *`target`* 的那 **两个** 整数，并返回它们的数组下标。你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。你可以按任意顺序返回答案。

分析：使用哈希表记录 `{nums[i], i}`，后续遍历中在哈希表中查找 `target - nums[i])`，若找到则获取其下标值

注：由于需要返回下标值，故利用哈希表 `value` 记录下标；若仅需判断是否存在，使用集合记录 `nums[i]` 即可

复杂度：`O(n)` `O(n)`

```java
public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> hashtable = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        if (hashtable.containsKey(target - nums[i])) {
            return new int[]{hashtable.get(target - nums[i]), i};
        }
        hashtable.put(nums[i], i);
    }
    return new int[0];
}
```

## [罗马数字转整数](https://leetcode.cn/problems/roman-to-integer/)(13)

问题：罗马数字包含以下七种字符: `I`， `V`， `X`， `L`，`C`，`D` 和 `M`。

```
字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000
```

例如， 罗马数字 `2` 写做 `II` ，即为两个并列的 1 。`12` 写做 `XII` ，即为 `X` + `II` 。 `27` 写做 `XXVII`, 即为 `XX` + `V` + `II` 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 `IIII`，而是 `IV`。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 `IX`。这个特殊的规则只适用于以下六种情况：

- `I` 可以放在 `V` (5) 和 `X` (10) 的左边，来表示 4 和 9。
- `X` 可以放在 `L` (50) 和 `C` (100) 的左边，来表示 40 和 90。 
- `C` 可以放在 `D` (500) 和 `M` (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。

分析：利用哈希表存储罗马字符和数值的对应关系，从左到右遍历，注意到**当前仅当特殊组合时第一位值小于第二位**，故只需判断当前位值 `value` 与下一位的大小，若小于则按特殊组合处理 `ans -= value`，否则 `ans += value`

复杂度：`O(n)` `O(1)`

```java
Map<Character, Integer> symbolValues = new HashMap<Character, Integer>() {{  // 创建匿名子类对象初始化
    put('I', 1);
    put('V', 5);
    put('X', 10);
    put('L', 50);
    put('C', 100);
    put('D', 500);
    put('M', 1000);
}};

public int romanToInt(String s) {
    int ans = 0;
    int n = s.length();
    for (int i = 0; i < n; i++) {  // 从高位（左）到低位（右）遍历
        int value = symbolValues.get(s.charAt(i));
        if (i < n - 1 && value < symbolValues.get(s.charAt(i + 1))) { // 当前位值<下一位值，表明是一个特殊组合
            ans -= value; // 特殊组合需要减去第一位的值
        } else {
            ans += value;
        }
    }
    return ans;
}
```

## [字母异位词分组](https://leetcode.cn/problems/group-anagrams/)(49)

问题：给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。**字母异位词** 是由重新排列源单词的所有字母得到的一个新单词。

分析：利用哈希表，找到对应相同字母异位词的若干 `str` 的共同特征作为 `key`，并将若干 `str`本身作为 `value`

解法1：**排序作为key**：由于互为字母异位词的两个字符串包含的字母相同，因此对两个字符串分别进行排序之后得到的字符串一定是相同的，故可以将排序之后的字符串作为哈希表的键。

复杂度：`O(Nklogk)` `O(Nk)`

```java
public List<List<String>> groupAnagrams(String[] strs) {
    Map<String, List<String>> hashMap = new HashMap<>();
    for (String str : strs) {
        char[] array = str.toCharArray();  // 先将字符串转为字符数组（方便处理排序）
        Arrays.sort(array);
        String key = new String(array);  // 不能用 char[] 作为key，因为不同数组的地址一定不同，故再转回String
        List<String> list = hashMap.getOrDefault(key, new ArrayList<>());  // 拿到哈希表对应的一组值，否则创建一个空列表
        list.add(str);  // 将字母异位词组合在一起
        hashMap.put(key, list);  // 更新哈希表
    }
    return new ArrayList<>(hashMap.values());  // 将哈希表中的值存入列表作为最终答案
}
```

解法2：**计数作为key**：由于互为字母异位词的两个字符串包含的字母相同，因此两个字符串中的相同字母出现的次数一定是相同的，故可以将每个字母出现的次数使用字符串表示，作为哈希表的键。

复杂度：`O(Nk)` `O(Nk)`

```java
public List<List<String>> groupAnagrams2(String[] strs) {
    Map<String, List<String>> hashMap = new HashMap<>();
    for (String str : strs) {
        int[] counts = new int[26];  // 由于字符串只包含小写字母，因此对于每个字符串，可以使用长度为26的数组记录每个字母出现的次数。
        int n = str.length();
        for (int i = 0; i < n; i++) {  // 记录每个字母出现的次数
            counts[str.charAt(i) - 'a']++;
        }
        StringBuffer sb = new StringBuffer();  // 将每个出现次数大于 0 的字母和出现次数按顺序拼接成字符串，作为哈希表的键
        for (int i = 0; i < 26; i++) {  // 这里按顺序遍历字母，相当于对字母排序了一遍
            if (counts[i] != 0) {
                sb.append((char) ('a' + i));  // 拼接字母
                sb.append(counts[i]);  // 拼接字母的出现次数
            }
        }
        String key = sb.toString();  // 不能用 StringBuffer 作为key，因为不同对象的地址一定不同，故再转回String
        List<String> list = hashMap.getOrDefault(key, new ArrayList<>());  // 拿到哈希表对应的一组值，否则创建一个空列表
        list.add(str);  // 将字母异位词组合在一起
        hashMap.put(key, list);  // 更新哈希表
    }
    return new ArrayList<>(hashMap.values());  // 将哈希表中的值存入列表作为最终答案
}
```

## [LRU 缓存](https://leetcode.cn/problems/lru-cache/)(146)

问题：请你设计并实现一个满足 [LRU (最近最少使用) 缓存](https://baike.baidu.com/item/LRU) 约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1` 。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value` ；如果不存在，则向缓存中插入该组 `key-value` 。如果插入操作导致关键字数量超过 `capacity` ，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 必须以 `O(1)` 的平均时间复杂度运行。

分析：通过 **哈希表 + 双向链表** 维护所有缓存中的 `k-v` 对：首先使用哈希表定位缓存项在双向链表中的位置，再将其移动到双向链表头部

* 双向链表按照被**使用的顺序**存储了这些 `k-v` 对，靠近头部的 `k-v` 对是**最近**使用的，而靠近尾部的 `k-v` 对是**最久未使用**的。
* 哈希表通过缓存数据的 `key` 映射到其在双向链表中的位置。

双向链表使用**伪头部** `dummyHead` 和**伪尾部** `dummyTail` 标记界限，使得添加节点和删除节点的时无需检查相邻的节点是否存在。

复杂度：`O(1)` `O(capacity)`

```java
class LRUCache {
    Map<Integer, Node> map = new HashMap(); // hashmap+双向链表 映射k-v对
    int capacity; // 最大容量
    int size; // 当前容量
    Node dummyHead; // 虚拟头节点（辅助头插）
    Node dummyTail; // 虚拟尾节点（辅助尾删）

    public LRUCache(int capacity) {
        this.capacity = capacity;
        this.size = 0;
        this.dummyHead = new Node(-1, -1);
        this.dummyTail = new Node(-1, -1);
        dummyHead.next = dummyTail; // 连接虚拟头节点和虚拟尾节点
        dummyTail.pre = dummyHead;
    }
    
    public int get(int key) { // O(1)
        Node tmp = map.get(key); // 先判断是否存在
        if(tmp != null){ // 存在
            delete(tmp); // 先删除
            insertToHead(tmp); // 再头插法
            return tmp.value;
        }
        return -1; // 不存在，返回-1
    }
    
    public void put(int key, int value) { // O(1)
        Node tmp = map.get(key); // 先判断是否存在
        if(tmp != null){ // 存在
            delete(tmp); // 先删除
            tmp.value = value; // 修改value
            insertToHead(tmp); // 再头插法
        }else{ // 不存在
            Node newNode = new Node(key, value); // 创建新节点准备插入
            insertToHead(newNode); // 头插法插入
            map.put(key, newNode); // 同时更新哈希表
            if(++size > capacity){ // 再判断是否超过容量，超过则删除尾节点
                Node tail = dummyTail.pre; // 利用虚拟尾节点快速定位并删除
                delete(tail);
                map.remove(tail.key); // 同时更新哈希表
                size--;
            }
        }
    }

    public void delete(Node tmp){ // 利用双向链表快速删除节点
        tmp.pre.next = tmp.next; // 前一个节点指向后一个节点
        tmp.next.pre = tmp.pre; // 后一个节点指向前一个节点
    }

    public void insertToHead(Node tmp){ // 将节点使用头插法（虚拟头节点作辅助）
        tmp.next = dummyHead.next; // 先连接后面的节点，因为dummyHead指针在前面，而后面的需要利用前面
        dummyHead.next.pre = tmp;
        dummyHead.next = tmp; // 后面的连接完成后，对前面的指针没有影响
        tmp.pre = dummyHead;
    }

    class Node{ // 定义双向链表
        int key;
        int value;
        Node pre;
        Node next;
        public Node(int key, int value){
            this.key = key;
            this.value = value;
        }
    }
}
```
